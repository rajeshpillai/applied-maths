    Vector Algebra (Class 12, Chapter 10)

    A written tutorial covering the NCERT Class 12 chapter on vector algebra: scalar vs vector quantities, position vectors and direction cosines, the basic operations (addition, subtraction, scalar multiplication), the dot product, the cross product, projections, and worked NCERT-style problems.

    Vectors are the mathematical objects that carry both magnitude and direction. Almost every physical quantity that involves “where” or “which way” — displacement, velocity, force, magnetic field — is a vector. This chapter formalises the algebra they obey.

    1. Scalars vs vectors

    1.1 The everyday distinction

    Question 1: “What is your height?” — Answer: 1.6 m. Just a number with a unit.

    Question 2: “How will you kick the ball to get it to your teammate?” — Answer: 5 N, toward the north. Both magnitude (5 N) and direction (north).

    Quantities of the first kind are scalars. Quantities of the second kind are vectors.

    1.2 Definitions

    • Scalar quantity: completely specified by its magnitude alone. Examples: mass, time, volume, density, speed, temperature.
    • Vector quantity: requires both magnitude and direction. Examples: displacement, velocity, acceleration, force, momentum, weight.

    1.3 Telling them apart by units

    Sometimes the question gives a numerical value with units and asks you to classify. Examples:

    Quantity Reason Type
    5 seconds seconds = unit of time scalar
    1000 cm³ unit of volume scalar
    10 N unit of force vector
    30 km/h speed (no direction given) scalar
    30 km/h toward north velocity (direction given) vector
    10 g/cm³ density scalar
    20 m/s toward north velocity vector
    40° an angle scalar

    The trap: 30 km/h is speed (scalar), but 30 km/h north is velocity (vector). Mass is scalar; weight (mass × g, directed toward Earth’s centre) is vector.

    2. Representing a vector

    A vector is drawn as a directed line segment — an arrow. The length represents the magnitude, the arrowhead points in the direction.

    2.1 Notation

    A vector is written with an arrow on top: →a, or in bold **a**. The magnitude is |→a| or just a.

    In coordinates (3-D), a vector is

    →a = a₁ î + a₂ ĵ + a₃ k̂

    where î, ĵ, k̂ are the unit vectors along the positive x-, y-, z-axes. The numbers a₁, a₂, a₃ are the components of →a.

    2.2 Magnitude

    |→a| = √(a₁² + a₂² + a₃²)

    2.3 Special vectors

    • Zero vector →0: magnitude 0, no defined direction. Components (0, 0, 0).

    • Unit vector: a vector of magnitude 1. Given any non-zero →a, its unit vector is

      â = →a / |→a|

      The hat â denotes the unit vector in the same direction as →a.

    • Position vector: the vector from the origin to a point P(x, y, z). It is →OP = x î + y ĵ + z k̂.

    2.4 Types of vectors

    Name Definition
    Equal Same magnitude AND same direction (regardless of position).
    Negative (of →a) Same magnitude but opposite direction. Written −→a.
    Like / unlike Like: same direction. Unlike: opposite directions.
    Collinear / parallel Acting along the same line (or parallel lines).
    Coplanar Lying in the same plane.

    2.5 Direction cosines and direction ratios

    If →a = a₁ î + a₂ ĵ + a₃ k̂ makes angles α, β, γ with the positive x-, y-, z-axes, then

    cos α = a₁ / |→a|,    cos β = a₂ / |→a|,    cos γ = a₃ / |→a|

    These are the direction cosines of →a, satisfying cos² α + cos² β + cos² γ = 1. The components themselves (or any non-zero multiples) are the direction ratios.

    3. Vector addition

    Two methods, geometrically equivalent.

    3.1 Triangle law

    Place →b so its tail coincides with the head of →a. The vector from →a’s tail to →b’s head is →a + →b.

    3.2 Parallelogram law

    Place →a and →b so their tails coincide. Complete the parallelogram. The diagonal from the common tail is →a + →b.

    3.3 Component-wise

    If →a = a₁î + a₂ĵ + a₃k̂ and →b = b₁î + b₂ĵ + b₃k̂, then

    →a + →b = (a₁ + b₁) î + (a₂ + b₂) ĵ + (a₃ + b₃) k̂

    3.4 Subtraction

    →a − →b = →a + (−→b), with components (a_i − b_i).

    3.5 Properties

    • Commutative: →a + →b = →b + →a.
    • Associative: (→a + →b) + →c = →a + (→b + →c).
    • Zero vector: →a + →0 = →a.
    • Negative: →a + (−→a) = →0.

    4. Scalar multiplication

    For a scalar k and a vector →a:

    k →a = k a₁ î + k a₂ ĵ + k a₃ k̂

    The magnitude is |k →a| = |k| · |→a|. The direction is the same as →a if k > 0, opposite if k < 0.

    4.1 Properties

    • k(→a + →b) = k→a + k→b
    • (k + l)→a = k→a + l→a
    • (kl)→a = k(l→a)
    • 1 · →a = →a and 0 · →a = →0

    5. The dot product (scalar product)

    For two vectors →a and →b making an angle θ (with 0 ≤ θ ≤ π):

    →a · →b = |→a| · |→b| · cos θ

    5.1 Component-wise formula

    →a · →b = a₁ b₁ + a₂ b₂ + a₃ b₃

    Multiply matching components and add.

    5.2 Why “scalar product”

    The result →a · →b is a scalar (a real number), not another vector. This is the source of the name.

    5.3 Properties

    Property Statement
    Commutative →a · →b = →b · →a
    Distributive →a · (→b + →c) = →a · →b + →a · →c
    Scalar (k→a) · →b = k(→a · →b) = →a · (k→b)
    →a · →a = |→a|² (squared magnitude)
    Zero vector →a · →0 = 0

    5.4 The orthogonality test

    →a ⊥ →b   ⇔   →a · →b = 0       (for non-zero vectors)

    This is the most-used consequence: to check if two vectors are perpendicular, compute their dot product and see if it’s zero.

    5.5 The standard basis dot products

    î · î = ĵ · ĵ = k̂ · k̂ = 1
î · ĵ = ĵ · k̂ = k̂ · î = 0

    These are what make the component-wise formula work — different basis vectors are mutually perpendicular, same basis vector gives 1.

    5.6 Angle between two vectors

    Solve the dot-product equation for cos θ:

    cos θ = (→a · →b) / (|→a| · |→b|)

    Example. |→a| = 3, |→b| = 2, →a · →b = 6. Then

    cos θ = 6 / (3 · 2) = 1     ⇒   θ = 0

    (The vectors are parallel in the same direction.)

    Example. |→a| = 3, |→b| = 2, →a · →b = 6 / cos⁻¹(1/2). Then cos θ = 1/2, so θ = π/3.

    5.7 Component example

    →a = î − 2ĵ + 3k̂, →b = 3î − 2ĵ + k̂. Then

    →a · →b = (1)(3) + (−2)(−2) + (3)(1) = 3 + 4 + 3 = 10

    6. Projection of one vector onto another

    The projection of →a onto →b (a vector along →b’s direction):

    proj_→b →a = (→a · →b̂) · →b̂ = ((→a · →b) / |→b|²) · →b

    The scalar projection (just the length, signed):

    scalar proj_→b →a = →a · →b / |→b|

    The projection vector has magnitude |→a| cos θ and lies along →b’s direction (or opposite, if the dot product is negative).

    6.1 Worked example

    Project →a = î + 3ĵ + 7k̂ onto →b = 7î − ĵ + 8k̂.

    →a · →b = (1)(7) + (3)(−1) + (7)(8) = 7 − 3 + 56 = 60
|→b| = √(49 + 1 + 64) = √114

    Scalar projection = 60/√114.

    The projection vector is (60/114) · →b = (60/114)(7î − ĵ + 8k̂).

    7. The cross product (vector product)

    For two vectors →a and →b making an angle θ (with 0 ≤ θ ≤ π):

    →a × →b = |→a| · |→b| · sin θ · n̂

    where n̂ is the unit vector perpendicular to both →a and →b, with direction given by the right-hand rule.

    7.1 The right-hand rule

    Point the fingers of your right hand from →a to →b (curling through the smaller angle between them). Your thumb points in the direction of →a × →b.

    7.2 The result is a vector

    Unlike the dot product, the cross product yields another vector, perpendicular to both inputs.

    7.3 Component-wise formula

    →a × →b = | î   ĵ   k̂  |
          | a₁  a₂  a₃ |
          | b₁  b₂  b₃ |

    Expanded:

    →a × →b = (a₂ b₃ − a₃ b₂) î − (a₁ b₃ − a₃ b₁) ĵ + (a₁ b₂ − a₂ b₁) k̂

    7.4 Properties

    Property Statement
    Anti-commutative →a × →b = −(→b × →a) (order matters!)
    Distributive →a × (→b + →c) = →a × →b + →a × →c
    Scalar (k→a) × →b = k(→a × →b) = →a × (k→b)
    →a × →a = →0 (cross product of any vector with itself is zero)
    Zero vector →a × →0 = →0

    7.5 The parallelism test

    →a ∥ →b   ⇔   →a × →b = →0    (for non-zero vectors)

    Two non-zero vectors are parallel iff their cross product is the zero vector.

    7.6 The standard basis cross products

    î × ĵ = k̂,     ĵ × k̂ = î,     k̂ × î = ĵ
ĵ × î = −k̂,    k̂ × ĵ = −î,    î × k̂ = −ĵ
î × î = ĵ × ĵ = k̂ × k̂ = →0

    (Mnemonic: cycle î → ĵ → k̂ → î — each pair gives the next, going against the cycle introduces a minus sign.)

    7.7 Area applications

    • The magnitude |→a × →b| equals the area of the parallelogram spanned by →a and →b.
    • Half of this gives the area of the triangle formed by them: (1/2) |→a × →b|.

    7.8 Worked example

    →a = 3î + ĵ + 4k̂, →b = î − ĵ + k̂.

    →a × →b = | î   ĵ   k̂ |
          | 3   1   4 |
          | 1  −1   1 |

      = î · (1·1 − 4·(−1)) − ĵ · (3·1 − 4·1) + k̂ · (3·(−1) − 1·1)
      = î · (1 + 4) − ĵ · (3 − 4) + k̂ · (−3 − 1)
      = 5î + ĵ − 4k̂

    Area of the parallelogram = |5î + ĵ − 4k̂| = √(25 + 1 + 16) = √42.

    8. Comparison: dot vs cross

    Property →a · →b →a × →b
    Result type scalar vector
    Formula |→a||→b| cos θ |→a||→b| sin θ · n̂
    Commutative? yes no (anti-commutative)
    Zero when vectors perpendicular vectors parallel

    9. Worked NCERT-style problems

    9.1 Find the angle

    |→a| = 3, |→b| = 2, →a · →b = √6. Find the angle.

    cos θ = √6 / (3 · 2) = √6/6 = 1/√6
θ = cos⁻¹(1/√6)

    9.2 Unit vector

    Find the unit vector in the direction of →a = 2î + 3ĵ + k̂.

    |→a| = √(4 + 9 + 1) = √14
â = (1/√14)(2î + 3ĵ + k̂)

    9.3 Vector joining two points

    Vector from point P(2, 3, 0) to Q(−1, 1, 2) is

    →PQ = (−1 − 2)î + (1 − 3)ĵ + (2 − 0)k̂ = −3î − 2ĵ + 2k̂

    Magnitude |→PQ| = √(9 + 4 + 4) = √17.

    9.4 Section formula

    If P divides →AB in the ratio m : n internally:

    →OP = (m · →OB + n · →OA) / (m + n)

    For external division, replace n with −n.

    9.5 Perpendicular vectors

    Find λ so that →a = 2î + λĵ + k̂ and →b = î − 2ĵ + 3k̂ are perpendicular.

    →a · →b = 0
(2)(1) + (λ)(−2) + (1)(3) = 0
2 − 2λ + 3 = 0
λ = 5/2

    9.6 Area of a triangle

    Vertices A(1, 2, 3), B(2, −1, 4), C(4, 5, −1). Find the area.

    Vectors →AB = (1, −3, 1) and →AC = (3, 3, −4).

    →AB × →AC = | î   ĵ   k̂ |
            | 1  −3   1 |
            | 3   3  −4 |

         = î((−3)(−4) − (1)(3)) − ĵ((1)(−4) − (1)(3)) + k̂((1)(3) − (−3)(3))
         = î(12 − 3) − ĵ(−4 − 3) + k̂(3 + 9)
         = 9î + 7ĵ + 12k̂

|→AB × →AC| = √(81 + 49 + 144) = √274

Area = (1/2)√274

    10. Common pitfalls

    • Confusing magnitude with the vector itself. |→a| is a scalar; →a is a vector.
    • Direction matters for vectors. Two vectors with the same length but pointing opposite ways are not equal.
    • Forgetting the right-hand rule. →a × →b ≠ →b × →a — getting the order wrong flips the sign.
    • î × ĵ = k̂, not î ĵ = k̂. The cross-product symbol matters.
    • Confusing zero scalar with zero vector. →a · →b = 0 (scalar zero); →a × →b = →0 (zero vector). Different objects.
    • Computing →a × →b and reading off a magnitude. The result is a vector, not a scalar. Take its magnitude if you want the area.
    • Forgetting that |→a × →b| = |→a||→b| sin θ uses the sine while →a · →b = |→a||→b| cos θ uses the cosine. Mixing these up is a common arithmetic error.

    Workflow:

    1. Express each vector in component form î, ĵ, k̂.
2. Identify the operation (sum, dot, cross, projection).
3. Apply the component-wise formula.
4. Simplify; take magnitude/scalar where needed.

    12. Looking ahead

    • Three-dimensional geometry (next chapter): the equations of lines and planes use vector algebra throughout. The angle between two lines, the foot of perpendicular, the distance from a point to a plane — all reduce to dot- and cross-products.
    • Mechanics. Force, velocity, acceleration, momentum, and angular momentum are all vector quantities. The torque →r × →F is a textbook application of the cross product.
    • Electricity and magnetism. Magnetic force on a moving charge →F = q→v × →B is built from a cross product. Gauss’s law uses dot products.
    • Computer graphics. Lighting, shading, rotations — every 3D rendering pipeline uses dot and cross products at its core.
    • Linear algebra. Vectors generalise to n-dimensional vector spaces. The dot product generalises to the inner product. The cross product is special to 3D (and 7D) but its role is taken over by more general operations.

    The single habit to internalise: whenever you face a vector problem, first decide what’s being asked geometrically — is it an angle? A perpendicular? An area? A projection? Each has a standard tool (dot, cross, or projection formula), and the algebra falls out from there.

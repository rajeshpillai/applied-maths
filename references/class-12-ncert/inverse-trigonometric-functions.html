    Inverse Trigonometric Functions (Class 12, Chapter 2)

    A written tutorial covering the NCERT Class 12 chapter on inverse trigonometric functions: why these functions need restricted domains, the six principal branches, their domains and ranges, graphs, and the standard identities used to manipulate them.

    This chapter pulls together three threads — trigonometry from Class 11, the idea of a one-one + onto function from Chapter 1 of this book, and the inversion procedure itself. It pays off to keep all three in mind.

    1. Why we need restricted domains

    A function f : A → B is invertible iff it is one-one and onto (Chapter 1). The ordinary trig functions are not one-one over their full domain — sin(0) = sin(π) = sin(2π) = 0, many different x give the same sin x. So sin : ℝ → ℝ has no inverse.

    The fix is to shrink the domain to an interval where the function is one-one, and shrink the codomain to its actual range. The standard choices for the six trig functions give the so-called principal value branches.

    2. The unit-circle picture

    A quick recap. Set θ as the angle measured anticlockwise from the positive x-axis, and let P(θ) be the point on the unit circle. By definition

    P(θ) = (cos θ, sin θ)

    So the x-coordinate of P(θ) is cos θ and the y-coordinate is sin θ. Reading off standard angles:

    θ cos θ sin θ
    0 1 0
    π/6 √3/2 1/2
    π/4 1/√2 1/√2
    π/3 1/2 √3/2
    π/2 0 1
    π −1 0
    3π/2 0 −1
    2π 1 0

    tan θ = sin θ / cos θ, cot θ = cos θ / sin θ, sec θ = 1/cos θ, cosec θ = 1/sin θ.

    3. Principal value branches

    For each of the six trig functions, the principal value branch is the longest convenient interval on which the function is one-one and onto its full range, with a preference for intervals close to 0.

    3.1 sine

    sin : [−π/2, π/2] → [−1, 1] is bijective. Its inverse is

    sin⁻¹ : [−1, 1] → [−π/2, π/2]

    Read sin⁻¹(x) as “the unique angle in [−π/2, π/2] whose sine is x”.

    3.2 cosine

    cos : [0, π] → [−1, 1] is bijective. Its inverse is

    cos⁻¹ : [−1, 1] → [0, π]

    3.3 tangent

    tan is not defined at odd multiples of π/2. The standard branch:

    tan : (−π/2, π/2) → ℝ is bijective. So

    tan⁻¹ : ℝ → (−π/2, π/2)

    3.4 cotangent

    cot : (0, π) → ℝ is bijective. So

    cot⁻¹ : ℝ → (0, π)

    3.5 secant

    sec : [0, π] \ {π/2} → ℝ \ (−1, 1) is bijective. So

    sec⁻¹ : ℝ \ (−1, 1) → [0, π] \ {π/2}

    3.6 cosecant

    cosec : [−π/2, π/2] \ {0} → ℝ \ (−1, 1) is bijective. So

    cosec⁻¹ : ℝ \ (−1, 1) → [−π/2, π/2] \ {0}

    3.7 Summary table

    Function Domain Range (principal branch)
    sin⁻¹ x [−1, 1] [−π/2, π/2]
    cos⁻¹ x [−1, 1] [0, π]
    tan⁻¹ x ℝ (−π/2, π/2)
    cot⁻¹ x ℝ (0, π)
    sec⁻¹ x (−∞, −1] ∪ [1, ∞) [0, π] \ {π/2}
    cosec⁻¹ x (−∞, −1] ∪ [1, ∞) [−π/2, π/2] \ {0}

    Memorise these. Every problem in this chapter depends on landing in the right branch.

    3.8 Notation warning

    sin⁻¹ x does not mean 1 / sin x. The “−1” is part of the function name, not a power. The reciprocal 1 / sin x is cosec x. Some textbooks write arcsin x to avoid confusion.

    4. Computing principal values

    The standard procedure: ask “what angle (in the right principal branch) gives this trig value?”

    4.1 Worked values

    Expression Reason Answer
    sin⁻¹(1/2) sin of π/6 is 1/2, and π/6 ∈ [−π/2, π/2] π/6
    sin⁻¹(−1/2) sin of −π/6 is −1/2, and −π/6 ∈ [−π/2, π/2] −π/6
    cos⁻¹(1/2) cos of π/3 is 1/2, and π/3 ∈ [0, π] π/3
    cos⁻¹(−1/2) cos of 2π/3 is −1/2, and 2π/3 ∈ [0, π] 2π/3
    tan⁻¹(1) tan of π/4 is 1, and π/4 ∈ (−π/2, π/2) π/4
    tan⁻¹(−√3) tan of −π/3 is −√3 −π/3
    sec⁻¹(2) sec of π/3 is 2 π/3
    cosec⁻¹(−2) cosec of −π/6 is −2 −π/6
    cot⁻¹(0) cot of π/2 is 0, and π/2 ∈ (0, π) π/2

    4.2 The “wrong” answer trap

    sin(2π/3) = √3/2, so it’s tempting to write sin⁻¹(√3/2) = 2π/3. Wrong — 2π/3 ∉ [−π/2, π/2]. The correct answer is π/3. Always check the branch.

    4.3 The negative-input rules

    For functions whose principal branch is symmetric around 0 (sin⁻¹, tan⁻¹, cosec⁻¹):

    sin⁻¹(−x) = −sin⁻¹(x)
tan⁻¹(−x) = −tan⁻¹(x)
cosec⁻¹(−x) = −cosec⁻¹(x)

    For functions whose principal branch starts at 0 (cos⁻¹, sec⁻¹, cot⁻¹):

    cos⁻¹(−x) = π − cos⁻¹(x)
sec⁻¹(−x) = π − sec⁻¹(x)
cot⁻¹(−x) = π − cot⁻¹(x)

    Quick check: cos⁻¹(−1/2) = π − cos⁻¹(1/2) = π − π/3 = 2π/3. ✓

    5. Graphs

    All six inverse trig graphs are obtained by reflecting the corresponding restricted-domain trig graph across the line y = x. Key features:

    • y = sin⁻¹ x: an S-shaped curve through the origin from (−1, −π/2) to (1, π/2). Increasing.
    • y = cos⁻¹ x: a curve from (−1, π) down to (1, 0). Decreasing.
    • y = tan⁻¹ x: an S-curve through the origin, with horizontal asymptotes at y = ±π/2. Strictly increasing on all of ℝ.
    • y = cot⁻¹ x: a curve decreasing from π (as x → −∞) to 0 (as x → ∞), passing through (0, π/2).
    • y = sec⁻¹ x and y = cosec⁻¹ x: two-piece graphs, split because the domain is split.

    These graphs are not just decoration — they encode at a glance which inverse trig values are positive/negative and where each function changes sign.

    6. Inverse-trig identities

    6.1 Inverse cancellations

    For x in the appropriate principal branch:

    sin(sin⁻¹ x) = x         for x ∈ [−1, 1]
cos(cos⁻¹ x) = x         for x ∈ [−1, 1]
tan(tan⁻¹ x) = x         for x ∈ ℝ

sin⁻¹(sin θ) = θ         only if θ ∈ [−π/2, π/2]
cos⁻¹(cos θ) = θ         only if θ ∈ [0, π]
tan⁻¹(tan θ) = θ         only if θ ∈ (−π/2, π/2)

    The f⁻¹(f(θ)) = θ direction is conditional: if θ is outside the principal branch, you must first reduce θ to its principal-branch representative.

    Example. sin⁻¹(sin(3π/4)) is not 3π/4 — that’s outside [−π/2, π/2]. But sin(3π/4) = sin(π − 3π/4) = sin(π/4). So sin⁻¹(sin(3π/4)) = π/4.

    6.2 Reciprocal identities

    For x in the appropriate domains:

    sin⁻¹(1/x) = cosec⁻¹(x)        for x ∈ (−∞, −1] ∪ [1, ∞)
cos⁻¹(1/x) = sec⁻¹(x)          for x ∈ (−∞, −1] ∪ [1, ∞)
tan⁻¹(1/x) = cot⁻¹(x)          for x > 0
tan⁻¹(1/x) = cot⁻¹(x) − π      for x < 0

    6.3 Complementary identities (the “π/2 identities”)

    The single most-used trio in problem-solving:

    sin⁻¹ x + cos⁻¹ x = π/2          for x ∈ [−1, 1]
tan⁻¹ x + cot⁻¹ x = π/2          for x ∈ ℝ
sec⁻¹ x + cosec⁻¹ x = π/2        for |x| ≥ 1

    The reasoning, illustrated for the first one: if sin θ = x with θ ∈ [−π/2, π/2], then cos(π/2 − θ) = sin θ = x, and π/2 − θ ∈ [0, π] so this is the right branch for cos⁻¹. Hence cos⁻¹(x) = π/2 − sin⁻¹ x.

    6.4 Sum and difference of tan⁻¹

    tan⁻¹ x + tan⁻¹ y = tan⁻¹((x + y)/(1 − xy))      if xy < 1
                  = π + tan⁻¹((x + y)/(1 − xy))   if xy > 1 and x, y > 0
                  = −π + tan⁻¹((x + y)/(1 − xy))  if xy > 1 and x, y < 0
    tan⁻¹ x − tan⁻¹ y = tan⁻¹((x − y)/(1 + xy))      if xy > −1

    The ±π adjustments come from making sure the result lands in the principal branch (−π/2, π/2). Most NCERT problems will set up xy < 1 so the simple form works.

    6.5 Double-argument identities

    2 tan⁻¹ x = sin⁻¹(2x / (1 + x²))           if |x| ≤ 1
          = cos⁻¹((1 − x²) / (1 + x²))     if x ≥ 0
          = tan⁻¹(2x / (1 − x²))           if |x| < 1

    These come from the standard tan(2θ), sin(2θ), cos(2θ) formulas. Each has a range condition because the right-hand inverse trig function has its own principal branch.

    6.6 Triple-argument identities

    3 sin⁻¹ x = sin⁻¹(3x − 4x³)        for |x| ≤ 1/2
3 cos⁻¹ x = cos⁻¹(4x³ − 3x)        for 1/2 ≤ x ≤ 1
3 tan⁻¹ x = tan⁻¹((3x − x³)/(1 − 3x²))   for |x| < 1/√3

    7. Worked problems

    7.1 Find the principal value

    tan⁻¹(−1).

    The angle in (−π/2, π/2) whose tan is −1 is −π/4.

    Answer: −π/4.

    7.2 Apply a complementary identity

    If sin⁻¹(1/5) + cos⁻¹(x) = 1 is impossible because there’s no choice of x that makes this exactly 1. The actual NCERT problem reads sin(sin⁻¹(1/5) + cos⁻¹ x) = 1, which means the argument of sin must be π/2. So

    sin⁻¹(1/5) + cos⁻¹ x = π/2
cos⁻¹ x = π/2 − sin⁻¹(1/5) = cos⁻¹(1/5)
x = 1/5

    7.3 Compose inside-out

    tan⁻¹(2 cos(2 sin⁻¹(1/2))).

    Innermost first: sin⁻¹(1/2) = π/6. Then 2 sin⁻¹(1/2) = π/3. Then cos(π/3) = 1/2. Then 2 · 1/2 = 1. Finally tan⁻¹(1) = π/4.

    7.4 Simplify a π/2 expression

    cot(tan⁻¹ a + cot⁻¹ a).

    By the complementary identity tan⁻¹ a + cot⁻¹ a = π/2. So the expression is cot(π/2) = 0.

    7.5 Double-argument application

    Simplify tan((1/2) [sin⁻¹(2x/(1 + x²)) + cos⁻¹((1 − y²)/(1 + y²))]).

    The first term inside is 2 tan⁻¹ x (by §6.5). The second is 2 tan⁻¹ y (by §6.5). So the bracket is 2 tan⁻¹ x + 2 tan⁻¹ y = 2(tan⁻¹ x + tan⁻¹ y). Multiply by 1/2: we get tan⁻¹ x + tan⁻¹ y. Now

    tan(tan⁻¹ x + tan⁻¹ y) = tan(tan⁻¹((x + y)/(1 − xy))) = (x + y)/(1 − xy)

    (assuming xy < 1).

    7.6 Out-of-branch value

    sin⁻¹(sin(2π/3)).

    2π/3 ∉ [−π/2, π/2], so we can’t just remove the inverse. Use sin(2π/3) = sin(π − 2π/3) = sin(π/3). So sin⁻¹(sin(2π/3)) = sin⁻¹(sin(π/3)) = π/3.

    7.7 Negative-input rule

    tan⁻¹(−x) + cot⁻¹(−x).

    By the complementary identity (which works for any real input), tan⁻¹ y + cot⁻¹ y = π/2 for any y. Take y = −x. Answer: π/2.

    7.8 Convert between inverse-trig forms

    Show tan⁻¹(x/a) = (1/3) tan⁻¹(3a²x − x³)/(a³ − 3ax²).

    Let θ = tan⁻¹(x/a). Then tan θ = x/a. Compute tan 3θ using the triple-angle formula:

    tan 3θ = (3 tan θ − tan³ θ)/(1 − 3 tan² θ)
       = (3(x/a) − (x/a)³) / (1 − 3(x/a)²)
       = (3a²x − x³) / (a³ − 3ax²)      (multiplying top and bottom by a³)

    So 3θ = tan⁻¹((3a²x − x³)/(a³ − 3ax²)), giving θ = (1/3) tan⁻¹(…). Done.

    8. Common pitfalls

    • sin⁻¹ x ≠ 1/sin x. The −1 is part of the function name. The reciprocal is cosec x.
    • Out-of-branch composition. sin⁻¹(sin θ) = θ only if θ ∈ [−π/2, π/2]. Outside that, reduce first.
    • Forgetting branch conditions on identities. 2 tan⁻¹ x = sin⁻¹(2x/(1+x²)) only when |x| ≤ 1. If |x| > 1, the right-hand side is in a different branch and you may need a π − … correction.
    • tan⁻¹ x + tan⁻¹ y formula’s ±π adjustment. If xy > 1, the sum is outside (−π/2, π/2) and needs an offset.
    • Confusing the negative-input rules. sin⁻¹(−x) = −sin⁻¹ x (odd-function-like), but cos⁻¹(−x) = π − cos⁻¹ x (different pattern).
    • Wrong-quadrant principal values. cos⁻¹(−1/2) = 2π/3, not −π/3. The principal branch of cos⁻¹ is [0, π], no negatives allowed.
    • Treating sec⁻¹ and cosec⁻¹ like the others. Their domains are |x| ≥ 1 only — you cannot take sec⁻¹(1/2).

    9. Quick reference card

    PRINCIPAL VALUE BRANCHES
  sin⁻¹ x     : [−1, 1]                 → [−π/2, π/2]
  cos⁻¹ x     : [−1, 1]                 → [0, π]
  tan⁻¹ x     : ℝ                       → (−π/2, π/2)
  cot⁻¹ x     : ℝ                       → (0, π)
  sec⁻¹ x     : ℝ \ (−1, 1)             → [0, π] \ {π/2}
  cosec⁻¹ x   : ℝ \ (−1, 1)             → [−π/2, π/2] \ {0}

NEGATIVE INPUT
  sin⁻¹(−x)   = −sin⁻¹ x
  tan⁻¹(−x)   = −tan⁻¹ x
  cosec⁻¹(−x) = −cosec⁻¹ x
  cos⁻¹(−x)   = π − cos⁻¹ x
  sec⁻¹(−x)   = π − sec⁻¹ x
  cot⁻¹(−x)   = π − cot⁻¹ x

COMPLEMENTARY ("π/2") IDENTITIES
  sin⁻¹ x + cos⁻¹ x   = π/2
  tan⁻¹ x + cot⁻¹ x   = π/2
  sec⁻¹ x + cosec⁻¹ x = π/2

RECIPROCAL IDENTITIES (in domain)
  sin⁻¹(1/x)  = cosec⁻¹ x         (|x| ≥ 1)
  cos⁻¹(1/x)  = sec⁻¹ x           (|x| ≥ 1)
  tan⁻¹(1/x)  = cot⁻¹ x           (x > 0)

SUM / DIFFERENCE OF tan⁻¹  (for xy < 1)
  tan⁻¹ x + tan⁻¹ y = tan⁻¹((x + y)/(1 − xy))
  tan⁻¹ x − tan⁻¹ y = tan⁻¹((x − y)/(1 + xy))

DOUBLE ANGLE
  2 tan⁻¹ x = sin⁻¹(2x/(1 + x²))       (|x| ≤ 1)
            = cos⁻¹((1 − x²)/(1 + x²)) (x ≥ 0)
            = tan⁻¹(2x/(1 − x²))       (|x| < 1)

    Standard solving recipe:

    1. Find the innermost inverse-trig term; reduce to a principal-value angle.
2. Apply trig (or use a sum/double-angle identity) to simplify.
3. Re-wrap with the outer inverse-trig if needed.
4. Check that your answer lies in the appropriate principal branch.

    10. Looking ahead

    • Differentiation of inverse trig functions. d/dx sin⁻¹ x = 1/√(1 − x²), d/dx tan⁻¹ x = 1/(1 + x²), etc. These are key building blocks for the next chapter.
    • Integration. Integrals of the form ∫ dx/√(1 − x²), ∫ dx/(1 + x²), ∫ dx/(x√(x² − 1)) evaluate directly to inverse trig functions.
    • Trigonometric substitution. Whenever an integrand has √(a² − x²), √(a² + x²), or √(x² − a²), an inverse-trig substitution (x = a sin θ, x = a tan θ, x = a sec θ) usually unlocks the problem.
    • Complex logarithm. Inverse trig functions can also be expressed as logarithms of complex numbers, e.g. sin⁻¹ z = −i log(iz + √(1 − z²)). This bridges the chapter to complex analysis.

    The single habit to internalise: every time you see sin⁻¹, cos⁻¹, tan⁻¹ (etc.) ask which principal branch? That single question stops 90% of the errors students make in this chapter.

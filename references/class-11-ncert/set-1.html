    Sets — Class 11 (NCERT Chapter 1)

    A complete, in-depth tutorial on the Sets chapter, based on a Hindi-medium one-shot lecture covering NCERT Maths 2025 syllabus. The chapter is the foundational language for nearly every later topic — Relations & Functions, Probability, Geometry, Algebra — so the goal here is not just to memorise definitions but to build the mental model that makes the rest of the year painless.

    1. What is a set?

    1.1 Why sets matter

    You can’t define relations or functions without sets. The same way you can’t divide without first knowing your multiplication tables, you can’t reason about a function f : A → B without knowing what the sets A and B are. Sets also reappear in geometry (a line is a set of points), algebra (the solution set of an equation), and probability (the sample space is a set).

    1.2 Collection vs. set

    In daily life we encounter many collections:

    • The rivers of India.
    • The natural numbers.
    • The 52 cards in a deck.
    • The DVDs on your shelf.

    Some of these can be sharpened into sets; others cannot. The line between the two is a single property: well-definedness.

    Definition. A set is a well-defined collection of objects.

    What does well-defined mean? Given any object, there must be an unambiguous yes/no answer to “is it in the collection?” — and crucially, everyone must agree on that answer.

    1.3 The vague-term trap

    The phrase “good cricket players” is not a set. Why? Because what counts as “good” is subjective — your list and mine can differ. The word “good” is a vague term.

    The same applies to:

    • Best cricket batsmen of the world.
    • Most talented writers of India.
    • Most dangerous animals of the world.

    These are collections, but they are not sets.

    By contrast, the following are sets:

    • Months of the year beginning with letter J → {January, June, July}. Anyone in any country gives the same answer.
    • All boys in your class. Even though we may not be able to list them, the membership is unambiguous.
    • All natural numbers less than 10 → {1, 2, 3, 4, 5, 6, 7, 8, 9}.
    • Novels written by Munshi Premchand. A finite, fixed list — not subjective.
    • All even integers.

    Rule of thumb: if the description contains a vague adjective (good, best, dangerous, talented), it’s not a set. Otherwise, it generally is.

    1.4 How to write the answer (avoid yes/no)

    In NCERT-style questions like “which of the following are sets?” the answer key may say “yes” or “no”, but the question asks you to justify. Always write:

    • If it is a set: The given collection consists of definite objects, therefore it is a set.
    • If it is not a set: The term [vague word] is a vague (not well-defined) term, therefore the given collection is not a set.

    2. Elements of a set

    The objects inside a set are called its elements. Two pieces of notation:

    Symbol Meaning Read as
    a ∈ A a is an element of A “a belongs to A”
    a ∉ A a is not an element of A “a does not belong to A”

    Convention. Sets are denoted by capital letters (A, B, C, X, Y, Z); their elements by lowercase letters.

    2.1 Worked example

    Let A = {1, 2, {3, 4}, 5}. Insert ∈ or ∉:

    Statement Answer Why
    5 _ A ∈ 5 is listed in A
    8 _ A ∉ 8 is not listed
    0 _ A ∉ 0 is not listed
    4 _ A ∉ 4 is inside the inner brace {3, 4}, not a top-level element
    2 _ A ∈ 2 is listed
    9 _ A ∉ 9 is not listed

    The fourth row is the trap: {3, 4} is a single element of A. The numbers 3 and 4 individually are not.

    3. Two ways to represent a set

    3.1 Roster (tabular) form

    Three rules:

    1. List all elements.
    2. Separate them by commas.
    3. Enclose them in curly braces { }.

    Example. Set A = “all natural numbers less than 10”:

    A = {1, 2, 3, 4, 5, 6, 7, 8, 9}

    Two important conventions:

    • Order doesn’t matter. {a, e, i, o, u} and {i, e, a, u, o} describe the same set.
    • Repetition has no meaning. The set of letters in SCHOOL is {S, C, H, O, L} — the second O is dropped.

    3.2 Set-builder form

    When the elements share a single property and listing them is impractical (or infinite), use:

    A = { x : P(x) }

    Read: “the set of all x such that P(x) holds.” The colon : is read as “such that”. (Some textbooks use | instead; both are equivalent.)

    Example. Vowels in the English alphabet:

    B = { x : x is a vowel in the English alphabet }

    This describes the same set as {a, e, i, o, u}.

    Worked example. {1, 2, 3, 4} in set-builder form:

    { x : x ∈ ℕ and x ≤ 4 }

    You must specify both the type (x ∈ ℕ) and the bound (x ≤ 4). Without x ∈ ℕ, x could be any real number ≤ 4, including −7 or 0.3.

    3.3 Three-dot notation

    When listing an infinite or large set, use exactly three dots …, never four or more. The three dots stand for “and so on”:

    {1, 3, 5, 7, …}        ✓ correct
{1, 3, 5, 7, …, 99}    ✓ correct (bounded above)
{1, 3, 5, 7, ......}   ✗ wrong (too many dots)

    3.4 Roster ↔︎ set-builder conversion drills

    Set-builder → roster:

    Given Roster form
    {x : x ∈ ℤ, −3 < x < 7} {−2, −1, 0, 1, 2, 3, 4, 5, 6}
    {x : x ∈ ℕ, x < 6} {1, 2, 3, 4, 5}
    {x : x is a 2-digit natural number, sum of digits = 8} {17, 26, 35, 44, 53, 62, 71, 80}
    {x : x is a prime divisor of 60} {2, 3, 5}
    Letters in word TRIGONOMETRY {T, R, I, G, O, N, M, E, Y}
    Letters in word BETTER {B, E, T, R}

    Roster → set-builder:

    Given Set-builder form
    {3, 6, 9, 12} { x : x = 3n, n ∈ ℕ, 1 ≤ n ≤ 4 }
    {2, 4, 8, 16, 32} { x : x = 2ⁿ, n ∈ ℕ, 1 ≤ n ≤ 5 }
    {5, 25, 125, 625} { x : x = 5ⁿ, n ∈ ℕ, 1 ≤ n ≤ 4 }
    {2, 4, 6, 8, …} { x : x = 2n, n ∈ ℕ }
    {1, 4, 9, 16, …, 100} { x : x = n², n ∈ ℕ, 1 ≤ n ≤ 10 }

    The pattern: identify the formula, name the parameter n, then specify both n ∈ ℕ and the bound on n.

    4. Types of sets

    4.1 Empty / null / void set

    Definition. A set with no elements is called the empty set, null set, or void set, denoted φ (phi) or { }.

    These three names are interchangeable.

    How an empty set arises in disguise:

    • A = { x : x ∈ ℕ, 1 < x < 2 } — no natural number lies strictly between 1 and 2, so A = φ.
    • B = { x : x is an even prime greater than 2 } — the only even prime is 2, so no element exists; B = φ.
    • C = { x : x is an odd natural number divisible by 2 } — impossible, C = φ.
    • D = { x : x ∈ ℕ, x < 5 and x > 7 } — no such natural number exists, D = φ.
    • E = { y : y is a point common to two parallel lines } — parallel lines never intersect, E = φ.

    Pitfall. {0} and {φ} are not empty. The first contains the number 0; the second contains one element (the symbol φ). The empty set is { } or φ written with no contents.

    4.2 Finite set

    Definition. A set is finite if it is empty or has a definite (countable) number of elements. Otherwise it is infinite.

    The number of elements in a finite set A is its cardinal number (or order), denoted n(A) or |A|.

    A = {2, 3, 4, 5, 6}    ⟹  n(A) = 5
B = {a, e, i, o, u}    ⟹  n(B) = 5

    4.3 Infinite set

    A set that is not finite — its elements cannot be exhaustively counted.

    ℕ = {1, 2, 3, 4, …}    ⟹ infinite
{ x : x is a multiple of 5 } ⟹ infinite
{ lines parallel to the x-axis } ⟹ infinite (you can draw infinitely many)
{ circles passing through the origin } ⟹ infinite

    4.4 Tricky finite/infinite examples

    Set Type Cardinal
    Months of the year Finite 12
    {1, 2, 3, …} Infinite —
    {1, 2, 3, …, 99, 100} Finite 100
    Positive integers > 100 Infinite —
    Prime numbers < 99 Finite (some specific value)
    Letters of English alphabet Finite 26
    Animals living on Earth Finite (some large finite number)

    The real-number trap. ℝ is infinite, but unlike ℕ it cannot even be expressed in roster form — there’s no recognisable pattern that captures every rational and every irrational. Always describe ℝ in set-builder form: { x : x is a real number }.

    4.5 Singleton set

    A set with exactly one element.

    {0}                          singleton
{ x : x is an even prime }   = {2}, singleton

    4.6 Equal sets

    Definition. Two sets A and B are equal (A = B) if they have exactly the same elements.

    Order doesn’t matter; repetition doesn’t matter. Only the underlying set of distinct elements counts.

    {1, 2, 3, 4}   =  {2, 1, 4, 3}    ✓ same elements
{1, 1, 2, 2, 3, 3} = {1, 2, 3}     ✓ duplicates ignored
{1, 2, 3}      ≠  {1, 2, 3, 5}     ✗ extra element
{1, 2, 3, 4}   ≠  {4, 8, 12, 16}   ✗ different elements

    Worked example. Are these equal?

    • A = { x : x is a positive even integer ≤ 10 } = {2, 4, 6, 8, 10}.
    • B = {2, 4, 6, 8, 10}.

    Yes, A = B.

    Trickier example. Are these equal?

    • A = { x : x is a solution of x² + 5x + 6 = 0 }.
    • B = {2, 3}.

    Solve x² + 5x + 6 = 0: factor as (x+2)(x+3) = 0, giving x = −2, −3. So A = {−2, −3}. Since A ≠ B, the answer is no.

    5. Subsets

    5.1 Definition and notation

    Definition. A is a subset of B (written A ⊆ B) if every element of A is also an element of B.

    If A ⊆ B and A ≠ B, we say A is a proper subset of B and write A ⊂ B.

    If A ⊆ B, then B is called the superset of A.

    Notation Meaning
    A ⊆ B every element of A is in B (allows A = B)
    A ⊂ B proper subset (every element in, but A ≠ B)
    A ⊄ B A is not a subset of B
    B ⊇ A B is a superset of A

    5.2 Three foundational properties

    Property 1. Every set is a subset of itself.

    A ⊆ A     for every set A

    This is because every element of A is trivially an element of A.

    Property 2. The empty set is a subset of every set.

    φ ⊆ A     for every set A

    The reasoning is vacuous truth: φ has no elements, so the statement “every element of φ is in A” is automatically true (no element exists to violate it).

    Property 3. Mutual subset implies equality.

    A ⊆ B  and  B ⊆ A    ⟹    A = B

    This is the standard technique for proving two sets are equal: prove each is a subset of the other.

    Conversely, if A ⊆ B but B ⊄ A, then A ≠ B (and A is a proper subset of B).

    5.3 Number of subsets formula

    Theorem. A finite set with n elements has exactly 2ⁿ subsets.

    Verification by counting:

    Set n Subsets Count
    φ 0 φ 1 = 2⁰
    {1} 1 φ, {1} 2 = 2¹
    {1, 2} 2 φ, {1}, {2}, {1,2} 4 = 2²
    {a, b, c} 3 φ, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c} 8 = 2³
    {1, 2, 3, 4} 4 (16 subsets) 16 = 2⁴

    Proof sketch (binomial argument). Choosing a subset of an n-element set is the same as making n independent yes/no decisions (“include element 1?”, “include element 2?”, …). That gives 2 × 2 × … × 2 = 2ⁿ total subsets. Equivalently, by the binomial theorem:

    ⁿC₀ + ⁿC₁ + ⁿC₂ + … + ⁿCₙ  =  (1+1)ⁿ  =  2ⁿ

    where ⁿCᵣ counts subsets of size exactly r. (The full proof from the binomial theorem is part of the Class 11 chapter on binomial expansions — accept the formula for now.)

    5.4 Element vs. subset — the most common mistake

    For A = {1, 2, {3, 4}, 5}:

    Statement True or false Why
    {3, 4} ∈ A True {3, 4} is listed as an element
    {3, 4} ⊆ A False for ⊆ we’d need 3 ∈ A and 4 ∈ A; neither holds
    {3, 4} ⊆ A True the singleton containing the element {3,4} is a subset
    1 ∈ A True
    1 ⊆ A False 1 is an element, not a set; subset is set-set relation
    {1} ⊆ A True {1} is a subset because 1 ∈ A
    {1, 2, 5} ⊆ A True each of 1, 2, 5 is in A
    {2, 3} ⊆ A False 3 is not in A (only {3,4} is)
    φ ∈ A False φ is not listed as an element
    φ ⊆ A True empty set is subset of every set
    {φ} ⊆ A False this would require φ ∈ A, which is false

    Mnemonic. To turn an element into a subset, wrap it in another pair of braces. 1 is an element ⟹ {1} is the corresponding singleton subset. The exception is φ: it’s a subset of every set without needing braces — but if φ is itself listed as an element of A, then {φ} becomes a singleton subset.

    6. Power set

    Definition. The power set of A, denoted P(A), is the set of all subsets of A.

    P(A) = { S : S ⊆ A }

    If A has n elements, then P(A) has 2ⁿ elements (one for each subset).

    6.1 Examples

    Example 1. A = {a, b}, n(A) = 2:

    P(A) = { φ, {a}, {b}, {a, b} }    →  n(P(A)) = 4

    Example 2. A = {1, 2, 3}, n(A) = 3:

    P(A) = { φ, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3} }    →  n(P(A)) = 8

    Example 3. A = φ, n(A) = 0:

    P(A) = { φ }    →  n(P(A)) = 1

    Crucial corollary. P(A) is never empty, even when A = φ. The empty set always counts as a subset, so P(A) always contains at least one element. Therefore: n(P(A)) ≥ 1 for every set A.

    6.2 Formula

    n(P(A)) = 2^{n(A)}

    7. Subsets of the real numbers

    Standard symbols you must memorise:

    Symbol Stands for Roster examples
    ℕ Natural numbers {1, 2, 3, 4, …}
    W Whole numbers {0, 1, 2, 3, …}
    ℤ Integers {…, −2, −1, 0, 1, 2, …}
    ℚ Rational numbers { p/q : p, q ∈ ℤ, q ≠ 0 }
    T Irrational numbers non-terminating, non-repeating decimals (√2, π, …)
    ℝ Real numbers rationals ∪ irrationals
    ℝ⁺ Positive real numbers
    ℝ⁻ Negative real numbers

    7.1 Inclusion chain

    Every natural number is a whole number; every whole number is an integer; every integer can be written as n/1, hence is rational; every rational is real:

    ℕ ⊂ W ⊂ ℤ ⊂ ℚ ⊂ ℝ

    Independently, T ⊂ ℝ and ℝ = ℚ ∪ T with ℚ ∩ T = φ.

    7.2 Real number is not roster-friendly

    Because ℝ mixes rationals and irrationals, and because between any two distinct reals there are infinitely many more, you cannot write ℝ as a list with three dots — no pattern is possible. Always use set-builder form for ℝ:

    ℝ = { x : x is a real number }

    8. Intervals as subsets of ℝ

    When a, b ∈ ℝ with a < b, an interval is a contiguous chunk of the real line between a and b. Four flavours, distinguished by which endpoint(s) are included:

    Notation Set-builder form Endpoints Type
    (a, b) { x : a < x < b } both excluded open interval
    [a, b] { x : a ≤ x ≤ b } both included closed interval
    [a, b) { x : a ≤ x < b } a included, b excluded left-closed / right-open
    (a, b] { x : a < x ≤ b } a excluded, b included left-open / right-closed

    8.1 Bracket convention

    • A round bracket ( or ) means the endpoint is excluded (no = in the inequality).
    • A square bracket [ or ] means the endpoint is included (≤ or ≥).

    8.2 Number-line representation

    • Excluded endpoint: open circle ○ at that point.
    • Included endpoint: filled circle ● at that point.
    (2, 5):   ────○════════○────
[2, 5]:   ────●════════●────
[2, 5):   ────●════════○────
(2, 5]:   ────○════════●────

    8.3 Worked conversions

    Interval Set-builder
    (−3, 0) { x : −3 < x < 0 }
    [6, 12] { x : 6 ≤ x ≤ 12 }
    (6, 12] { x : 6 < x ≤ 12 }
    [−23, 5) { x : −23 ≤ x < 5 }

    And the reverse:

    Set-builder Interval
    { x : −3 < x < 0 } (−3, 0)
    { x : 6 ≤ x ≤ 12 } [6, 12]
    { x : −1 ≤ x ≤ 10 } [−1, 10]
    { x : −23 < x < 5 } (−23, 5)

    9. Universal set

    Definition. A set U containing all the sets under discussion in a given context is called the universal set.

    The universal set acts as the “background” — every other set in the discussion is a subset of U.

    Example. Given A = {1, 2, 3}, B = {2, 3, 4}, C = {4, 5, 6}:

    • U = {1, 2, 3, 4, 5, 6} is a universal set.
    • U' = {1, 2, …, 10} also qualifies.
    • U'' = {1, 2, …, 100} is fine too.

    A universal set just has to contain all the elements of the sets you’re working with; it can have extras.

    9.1 Universal set vs. union (don’t confuse)

    A ∪ B ∪ C is the exact combination — every element of A, B, or C, no more.

    A universal set for A, B, C is any set that contains all those elements — it may have extras.

    9.2 Worked examples

    Question. Propose a universal set for the set of right triangles and the set of isosceles triangles.

    Answer. The set of all triangles. Every right triangle and every isosceles triangle is a triangle, so the set of all triangles contains both.

    Question. Which of the following is a universal set for A = {1, 3, 5}, B = {2, 4, 6}, C = {0, 2, 4, 6, 8}?

    1. {0, 1, 2, 3, 4, 5, 6} — missing 8 (in C). ✗
    2. φ — empty set has no elements. ✗
    3. {0, 1, 2, …, 10} — contains all elements of A, B, C. ✓
    4. {1, 2, 3, 4, 5, 6, 7, 8, 9} — missing 0 (in C). ✗

    Answer: (c).

    10. Operations on sets

    Five operations to master: union, intersection, difference, and complement, plus the special status of disjoint sets.

    10.1 Union (∪)

    Definition. A ∪ B is the set of all elements that are in A or in B (or both). Common elements appear only once.

    A ∪ B = { x : x ∈ A or x ∈ B }

    How to compute: combine all elements from both sets, dropping duplicates.

    Examples.

    A = {1, 2, 3, 4}, B = {2, 4, 5}    ⟹  A ∪ B = {1, 2, 3, 4, 5}
A = {a, b}, B = {c, d, e, f}        ⟹  A ∪ B = {a, b, c, d, e, f}
A = {1, 2, 3}, B = φ                ⟹  A ∪ B = A = {1, 2, 3}

    10.2 Property: union with subset

    If A ⊆ B, then A ∪ B = B.

    Reason. Every element of A is already in B, so combining them adds nothing new.

    Verification. A = {a, b}, B = {a, b, c} ⟹ A ⊆ B. A ∪ B = {a, b, c} = B. ✓

    10.3 Intersection (∩)

    Definition. A ∩ B is the set of all elements that belong to both A and B.

    A ∩ B = { x : x ∈ A and x ∈ B }

    How to compute: find the elements common to both sets.

    Examples.

    A = {1, 2, 3, 4}, B = {2, 3, 5}      ⟹  A ∩ B = {2, 3}
A = {a, b, c}, B = {b, c, d, e}       ⟹  A ∩ B = {b, c}
A = {1, 2, 3}, B = {6, 7, 8}          ⟹  A ∩ B = φ

    10.4 Iterated operations

    Both ∪ and ∩ are associative:

    A ∪ B ∪ C = (A ∪ B) ∪ C = A ∪ (B ∪ C)
A ∩ B ∩ C = (A ∩ B) ∩ C = A ∩ (B ∩ C)

    For three sets:

    A ∪ B ∪ C = elements in A or B or C  (drop duplicates)
A ∩ B ∩ C = elements in A and B and C

    10.5 Worked exercise

    Let

    A = {3, 5, 7, 9, 11}
B = {7, 9, 11, 13}
C = {11, 13, 15}
D = {15, 17}
    Expression Compute step by step Result
    A ∩ B common to A and B {7, 9, 11}
    B ∩ C common to B and C {11, 13}
    A ∩ C ∩ D common to all three φ
    A ∩ C common to A and C {11}
    B ∩ D common to B and D φ
    A ∩ (B ∪ C) first B ∪ C = {7,9,11,13,15}; intersect with A {7, 9, 11}
    (A ∩ B) ∪ (B ∩ C) {7,9,11} ∪ {11,13} {7, 9, 11, 13}
    A ∪ (B ∩ D) B ∩ D = φ; A ∪ φ = A {3, 5, 7, 9, 11}

    10.6 Property: intersection with subset

    If A ⊆ B, then A ∩ B = A.

    Reason. Every element of A is in B, so the elements common to both are exactly those of A.

    10.7 Disjoint sets

    Definition. Two sets A and B are disjoint if A ∩ B = φ — they share no element.

    Examples.

    A B Disjoint?
    {1, 2, 3, 4} {4, 5, 6} No (4 is common)
    {a, b, c, d, e} {e, f, g, h} No (e is common)
    even integers odd integers Yes
    vowels consonants Yes
    {2, 3, 4, 5} {6, 7, 8} Yes

    Standard write-up:

    • If disjoint: “Since A ∩ B = φ, A and B are disjoint sets.”
    • If not disjoint: “Since A ∩ B ≠ φ (here A ∩ B = {…}), A and B are not disjoint.”

    10.8 Difference (A − B)

    Definition. A − B is the set of elements in A but not in B.

    A − B = { x : x ∈ A and x ∉ B }

    Read aloud: “elements of A that are not in B.”

    Important: A − B and B − A are generally different.

    Examples. A = {1, 2, 3}, B = {2, 3, 4}, C = {3, 5, 7}.

    Expression Procedure Result
    A − B elements in A not in B {1}
    B − A elements in B not in A {4}
    A − C elements in A not in C {1, 2}
    C − A elements in C not in A {5, 7}
    B − C elements in B not in C {2, 4}
    C − B elements in C not in B {5, 7}

    10.9 Real-number example

    If ℝ is the set of real numbers and ℚ the rationals,

    ℝ − ℚ = T    (the set of irrational numbers)

    10.10 Complement (A' or Aᶜ)

    Definition. Given a universal set U, the complement of A (with A ⊆ U) is

    A' = U − A = { x : x ∈ U and x ∉ A }

    It’s the difference computed with U on the left.

    Example. U = {a, b, c, d, e, f, g, h}, A = {a, b, c}, B = {b, c, d}.

    A' = U − A = {d, e, f, g, h}
B' = U − B = {a, e, f, g, h}

    10.11 Properties of complement

    • (A')' = A — complement of complement is the original set.
    • A ∪ A' = U
    • A ∩ A' = φ
    • U' = φ and φ' = U.

    10.12 De Morgan’s laws (preview)

    For any two subsets of U:

    (A ∪ B)' = A' ∩ B'
(A ∩ B)' = A' ∪ B'

    In words: the complement of a union is the intersection of complements, and vice versa. These are the most-used identities in later set-algebra problems and probability.

    11. Visual summary — the operation table

    For two sets A and B inside a universal set U:

    Operation Result region (in a Venn diagram)
    A ∪ B everything in either circle
    A ∩ B only the overlap
    A − B the part of A’s circle outside B
    B − A the part of B’s circle outside A
    A' everything in U outside A’s circle
    (A ∪ B)' everything in U outside both circles
    (A ∩ B)' everything in U not in the overlap (most of U)

    12. Worked end-of-chapter problems

    12.1 Subset puzzle

    Q. Write all subsets of {1, 2, 3}.

    A. With n = 3, expect 2³ = 8 subsets:

    φ, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}

    12.2 Power set count

    Q. If A = φ, find n(P(A)).

    A. n(A) = 0, so n(P(A)) = 2⁰ = 1. Indeed P(A) = {φ}.

    12.3 Cardinal-number reasoning

    Q. A finite set A has 5 subsets that contain a particular element x. How many subsets does A have in total?

    A. Subsets that contain x correspond to subsets of A − {x} (just add x to each). So n(A) − 1 = log₂ 5 doesn’t divide evenly — something’s off; this means the question is ill-posed unless 5 is taken loosely. (In the actual NCERT formulation the number is 2^{n−1}, so if the answer is 2^{n−1} = ?, you’d back out n.)

    12.4 Difference + union combo

    Q. Given X = {a, b, c, d}, Y = {f, b, d, g}, find X − Y, Y − X, X ∩ Y.

    A.

    X − Y = {a, c}      (in X, not in Y)
Y − X = {f, g}      (in Y, not in X)
X ∩ Y = {b, d}      (common)

    Note that

    (X − Y) ∪ (X ∩ Y) ∪ (Y − X)  =  {a, c} ∪ {b, d} ∪ {f, g}  =  X ∪ Y

    This is the standard partition identity for two sets.

    14. Common mistakes — the master list

    1. Vague-term confusion. Writing “good,” “best,” “talented,” etc., into a set definition. Always check for vague adjectives.
    2. Repetition. Writing {1, 1, 2, 3}. Always reduce to {1, 2, 3}.
    3. Missing n ∈ ℕ clause in set-builder form. Without it, x = 2n could mean n = 1.5 and x = 3.
    4. Element vs. subset. Confusing 1 ∈ A with {1} ⊆ A. Element is bare; subset is wrapped in braces.
    5. φ vs. {φ}. φ has zero elements; {φ} is a singleton.
    6. P(φ) is not empty. It’s {φ}, with one element.
    7. A − B = B − A. False in general. Difference is not symmetric.
    8. Ordering matters in roster form. False — {1, 2} and {2, 1} are equal.
    9. Universal set = union. False. Universal set may have extras; union has exactly the combined elements.
    10. Three dots vs. four dots. Use exactly three dots …. Four-or-more dots have no formal meaning.

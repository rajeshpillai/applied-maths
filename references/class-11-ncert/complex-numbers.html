    Complex Numbers (Class 11, Chapter 5)

    A written tutorial covering the foundations of complex numbers as introduced in NCERT Class 11. The chapter title in the NCERT is Complex Numbers and Quadratic Equations; this tutorial focuses on the complex-number machinery (definition of i, algebraic form, four operations, conjugate, modulus, Argand plane, polar form) and ends with the connection back to quadratic equations.

    1. Why complex numbers exist

    The motivation comes from quadratic equations. Recall the quadratic formula for ax² + bx + c = 0:

    x = (-b ± √(b² − 4ac)) / (2a)

    The expression D = b² − 4ac is the discriminant. Class 10 already taught:

    Discriminant Roots
    D > 0 two distinct real roots
    D = 0 one repeated real root
    D < 0 “no real roots” — Class 10 stopped here

    When D < 0 we are forced to take the square root of a negative number, and Class 10 simply declared that no roots exist. Complex numbers are the upgrade that lets us continue.

    Worked motivation

    Take x² + x + 1 = 0. Here a = b = c = 1, so

    x = (−1 ± √(1 − 4)) / 2 = (−1 ± √(−3)) / 2

    We cannot finish this until we make sense of √(−3).

    2. The imaginary unit i (iota)

    Define a new symbol i by the rule

    i = √(−1)        ⇒        i² = −1

    With this single definition every square root of a negative real number can be rewritten:

    √(−n) = √n · √(−1) = √n · i        (for n > 0)

    Examples:

    • √(−2) = √2 · i
    • √(−3) = √3 · i
    • √(−7) = √7 · i

    Returning to x² + x + 1 = 0:

    x = (−1 ± √3 · i) / 2

    So the equation has two complex roots: x = (−1 + √3 i)/2 and x = (−1 − √3 i)/2.

    Powers of i — the four-cycle

    Direct computation:

    Power Value
    i¹ i
    i² −1
    i³ i² · i = −i
    i⁴ (i²)² = (−1)² = 1

    After i⁴ = 1, the cycle repeats: i⁵ = i, i⁶ = −1, i⁷ = −i, i⁸ = 1, …

    To compute any large power iⁿ, divide n by 4 and keep only the remainder r ∈ {0, 1, 2, 3}; then iⁿ = iʳ.

    Example. i⁹ + i¹⁹. Write i⁹ = (i⁴)² · i = 1·i = i and i¹⁹ = (i⁴)⁴ · i³ = 1·(−i) = −i. So i⁹ + i¹⁹ = i − i = 0.

    A small trap: √(−a)·√(−b) ≠ √(ab) when both are negative

    The identity √x · √y = √(xy) is valid for non-negative reals. Once one number is negative, you must convert to i first:

    √(−2) · √(−3) = (√2 · i)(√3 · i) = √6 · i² = −√6

    If you naively applied √(xy) you would get +√6, which is wrong.

    3. Algebraic form of a complex number

    Any complex number can be written

    z = a + ib       (a, b ∈ ℝ)
    • a is the real part, written Re(z).
    • b is the imaginary part, written Im(z).

    Note: Im(z) = b, not ib. Only the real coefficient is the imaginary part.

    A complex number is purely real if b = 0 and purely imaginary if a = 0 (with b ≠ 0).

    Equality of complex numbers

    a + ib = c + id if and only if a = c and b = d. Equating real parts to real parts and imaginary parts to imaginary parts is the workhorse trick of this chapter.

    Worked: bring expressions into a + ib form

    Example 1. 5i · (−3/5 i).

    Group reals together and i’s together: (5 · −3/5)(i · i) = −3 · i² = −3·(−1) = 3. So z = 3 + 0i.

    Example 2. Simplify i⁹ + i¹⁹. Already done above: 0.

    Example 3. (1 + i)⁴. Use (1+i)² = 1 + 2i + i² = 2i, then (2i)² = 4i² = −4. So z = −4 + 0i.

    4. Addition and subtraction

    Add real to real and imaginary to imaginary:

    (a + ib) + (c + id) = (a + c) + i(b + d)
(a + ib) − (c + id) = (a − c) + i(b − d)

    Example. z₁ = 2 + 3i, z₂ = 3 + 4i. Then z₁ + z₂ = 5 + 7i.

    Properties of addition

    Property Statement
    Closure sum of two complex numbers is a complex number
    Commutative z₁ + z₂ = z₂ + z₁
    Associative (z₁ + z₂) + z₃ = z₁ + (z₂ + z₃)
    Additive identity the zero complex number 0 + 0i satisfies z + 0 = z
    Additive inverse for z = a + ib, the inverse is −z = −a − ib; z + (−z) = 0

    Subtraction z₁ − z₂ is just z₁ + (−z₂), but in practice we subtract directly.

    5. Multiplication

    Multiply like binomials, then replace i² = −1:

    (a + ib)(c + id) = ac + iad + ibc + i²bd
                 = (ac − bd) + i(ad + bc)

    Example. (2 + 3i)(1 + 2i) = 2 + 4i + 3i + 6i² = 2 + 7i − 6 = −4 + 7i.

    Properties of multiplication

    Property Statement
    Closure product of two complex numbers is a complex number
    Commutative z₁ · z₂ = z₂ · z₁
    Associative (z₁z₂)z₃ = z₁(z₂z₃)
    Multiplicative identity 1 + 0i (i.e. the real number 1) satisfies z · 1 = z
    Distributive z₁(z₂ + z₃) = z₁z₂ + z₁z₃
    Multiplicative inverse for any z ≠ 0, there exists z⁻¹ such that z · z⁻¹ = 1

    The multiplicative inverse formula is derived in §7.

    6. Conjugate of a complex number

    For z = a + ib, the conjugate is

    z̄ = a − ib

    Only the sign of the imaginary part flips. Compare to the negative, which flips both signs:

    Operation Result
    z = 3 + 4i original
    −z = −3 − 4i both signs flipped
    z̄ = 3 − 4i only imaginary sign flipped

    For z = −2 − i, the conjugate is z̄ = −2 + i (the real part −2 stays).

    Key identities

    Let z = a + ib. Then:

    1. z + z̄ = 2a = 2 Re(z)
    2. z − z̄ = 2ib = 2i · Im(z)
    3. z · z̄ = a² + b² (a non-negative real number)
    4. z̄̄ = z (conjugating twice returns the original)

    Conjugate distributes over the four operations

    Identity
    (z₁ + z₂)¯ = z̄₁ + z̄₂
    (z₁ − z₂)¯ = z̄₁ − z̄₂
    (z₁ z₂)¯ = z̄₁ · z̄₂
    (z₁ / z₂)¯ = z̄₁ / z̄₂ (z₂ ≠ 0)

    A useful corollary: if z is a root of a polynomial with real coefficients, then z̄ is also a root. That is why complex roots of real quadratics always appear in conjugate pairs.

    7. Modulus and the multiplicative inverse

    Definition of modulus

    For z = a + ib,

    |z| = √(a² + b²)

    |z| is a non-negative real number; it measures the “size” of z. The squaring trick:

    |z|² = a² + b² = z · z̄

    This is the single most-used identity in the chapter.

    Example. z = 4 + 3i ⇒ |z| = √(16 + 9) = √25 = 5.

    Properties of modulus

    Property Statement
    |z| ≥ 0 with equality iff z = 0
    |z| = |z̄| = |−z|
    |z₁ z₂| = |z₁| · |z₂|
    |z₁ / z₂| = |z₁| / |z₂| (z₂ ≠ 0)
    |zⁿ| = |z|ⁿ
    Triangle inequality |z₁ + z₂| ≤ |z₁| + |z₂|

    Multiplicative inverse via the conjugate

    To find z⁻¹ = 1/z we rationalise by multiplying numerator and denominator by z̄:

    1 / (a + ib) = (a − ib) / ((a + ib)(a − ib))
            = (a − ib) / (a² + b²)
            = z̄ / |z|²

    So

    z⁻¹ = z̄ / |z|²

    Example. Inverse of 3 + 4i. |z|² = 9 + 16 = 25, z̄ = 3 − 4i, so z⁻¹ = (3 − 4i) / 25 = 3/25 − (4/25)i.

    8. Division of complex numbers

    Division is multiplication by the inverse, and in practice it is performed by multiplying numerator and denominator by the conjugate of the denominator.

    (a + ib) / (c + id) = (a + ib)(c − id) / ((c + id)(c − id))
                    = ((ac + bd) + i(bc − ad)) / (c² + d²)

    Example. (3 + 2i)/(1 − i). Multiply top and bottom by 1 + i:

    ((3 + 2i)(1 + i)) / ((1 − i)(1 + i)) = (3 + 3i + 2i + 2i²) / (1 + 1)
                                     = (3 − 2 + 5i) / 2
                                     = (1 + 5i)/2
                                     = 1/2 + (5/2)i

    Why we never leave i in the denominator

    The point of rationalising is exactly this: an answer like 1/(a + ib) is correct but not in standard form. By multiplying by (a − ib)/(a − ib) we expose (a + ib)(a − ib) = a² + b², a real number, which can sit safely in the denominator.

    9. Argand plane (geometric representation)

    A complex number z = x + iy is a pair of real numbers (x, y) and so can be plotted on a 2-D plane. Use two perpendicular axes:

    • horizontal axis for the real part — called the real axis;
    • vertical axis for the imaginary part — called the imaginary axis.

    This plane is the Argand plane (or complex plane).

                    imaginary axis
                       │
                       │  ● P(x, y)  ←— represents z = x + iy
                       │
       ────────────────┼──────────────── real axis
                       │
                       │
    Type of z Lies on / in
    z = a purely real (b = 0) the real axis
    z = ib purely imaginary (a = 0) the imaginary axis
    Re(z) > 0, Im(z) > 0 first quadrant
    Re(z) < 0, Im(z) > 0 second quadrant
    Re(z) < 0, Im(z) < 0 third quadrant
    Re(z) > 0, Im(z) < 0 fourth quadrant

    Geometric meaning of conjugate, negative, and modulus

    • z̄ = x − iy is the reflection of z in the real axis.
    • −z = −x − iy is the reflection of z through the origin.
    • |z| = √(x² + y²) is the distance from the origin to the point z.
    • |z₁ − z₂| is the distance between the points z₁ and z₂.

    10. Polar (trigonometric) form

    Let z = x + iy ≠ 0 correspond to point P(x, y). Let r = |z| (distance from the origin) and let θ be the angle the segment OP makes with the positive real axis (measured anticlockwise).

    x = r cos θ
y = r sin θ

    Substituting:

    z = r(cos θ + i sin θ)

    This is the polar form. Here

    • r = |z| = √(x² + y²) is the modulus,
    • θ = arg(z) is the argument (or amplitude) of z.

    Principal argument

    For a non-zero z, infinitely many angles describe the same point (any θ + 2πk). The unique value in (−π, π] is the principal argument, written Arg(z).

    To find θ, first compute the reference angle α = arctan(|y|/|x|), then place θ in the correct quadrant:

    Location of (x, y) Principal argument
    First quadrant (x > 0, y > 0) α
    Second quadrant (x < 0, y > 0) π − α
    Third quadrant (x < 0, y < 0) −(π − α) (or α − π)
    Fourth quadrant (x > 0, y < 0) −α

    Special cases: x > 0, y = 0 ⇒ 0; x = 0, y > 0 ⇒ π/2; x < 0, y = 0 ⇒ π; x = 0, y < 0 ⇒ −π/2.

    Worked example

    Express z = −1 + i in polar form.

    • |z| = √(1 + 1) = √2.
    • (−1, 1) lies in the second quadrant; reference angle α = arctan(1/1) = π/4.
    • Principal argument θ = π − π/4 = 3π/4.

    So z = √2 (cos 3π/4 + i sin 3π/4).

    Multiplication and division in polar form

    If z₁ = r₁(cos θ₁ + i sin θ₁) and z₂ = r₂(cos θ₂ + i sin θ₂), then

    z₁ z₂ = r₁ r₂ · (cos(θ₁ + θ₂) + i sin(θ₁ + θ₂))
z₁ / z₂ = (r₁ / r₂) · (cos(θ₁ − θ₂) + i sin(θ₁ − θ₂))

    Multiplication multiplies moduli and adds arguments; division divides moduli and subtracts arguments.

    11. Worked problem set

    Each of these is a small, isolated exercise reinforcing one technique.

    11.1 Bring into a + ib form

    Simplify (5 + 3i)(2 − i).

    (5 + 3i)(2 − i) = 10 − 5i + 6i − 3i²
               = 10 + i + 3
               = 13 + i

    11.2 Conjugate and modulus

    Find the conjugate and modulus of z = 4 + 3i.

    z̄ = 4 − 3i, |z| = √(16 + 9) = 5.

    11.3 Multiplicative inverse

    Find z⁻¹ for z = 4 − 3i.

    |z|² = 16 + 9 = 25, z̄ = 4 + 3i, so z⁻¹ = (4 + 3i)/25 = 4/25 + (3/25)i.

    11.4 Division

    Simplify (3 − 2i) / (2 + i).

    Multiply top and bottom by 2 − i:

    (3 − 2i)(2 − i) / ((2 + i)(2 − i)) = (6 − 3i − 4i + 2i²) / (4 + 1)
                                  = (6 − 7i − 2) / 5
                                  = (4 − 7i)/5
                                  = 4/5 − (7/5) i

    11.5 Prove a conjugate–modulus identity

    Show that if (x − iy) = √((a − ib)/(c − id)), then (x² + y²)² = (a² + b²)/(c² + d²).

    Take the equation as Eq. 1. Replacing i by −i throughout gives the conjugate equation:

    x + iy = √((a + ib)/(c + id))           (Eq. 2)

    Multiply Eq. 1 and Eq. 2:

    (x − iy)(x + iy) = √(((a − ib)(a + ib)) / ((c − id)(c + id)))
        x² + y² = √((a² + b²) / (c² + d²))

    Squaring both sides:

    (x² + y²)² = (a² + b²) / (c² + d²)

    11.6 Use |z|² = z · z̄

    Given |β| = 1, prove |(β − α) / (1 − α̅β)| = 1.

    Let w = (β − α) / (1 − α̅β). Then

    |w|² = w · w̄
     = ((β − α)/(1 − α̅β)) · ((β̄ − ᾱ)/(1 − α β̄))

    Multiply out the numerator and denominator and use β β̄ = |β|² = 1 everywhere it appears. The numerator and denominator both simplify to 1 + |α|² − αβ̄ − ᾱβ, so |w|² = 1, hence |w| = 1.

    11.7 Solve a power equation

    Find the smallest positive integer m such that ((1 + i)/(1 − i))ᵐ = 1.

    Rationalise the inside first:

    (1 + i)/(1 − i) = (1 + i)² / ((1 − i)(1 + i))
               = (1 + 2i + i²) / (1 + 1)
               = 2i / 2
               = i

    So we need iᵐ = 1. Since i⁴ = 1 and no smaller positive power equals 1, the answer is m = 4.

    11.8 Number of non-zero integral solutions

    Find the number of non-zero integer solutions of |1 − i|ˣ = 2ˣ.

    |1 − i| = √2. So (√2)ˣ = 2ˣ, i.e. 2^(x/2) = 2ˣ, giving x/2 = x, hence x = 0. Since the only solution is x = 0, the number of non-zero integer solutions is 0.

    12. Quadratic equations with complex roots

    The reason this chapter pairs complex numbers with quadratics in NCERT.

    For ax² + bx + c = 0 with real a, b, c:

    x = (−b ± √(b² − 4ac)) / (2a)

    When D = b² − 4ac < 0, write √D = √(−|D|) = i√|D|. The roots become

    x = (−b ± i√|D|) / (2a)

    These are a conjugate pair; calling them p + iq and p − iq, we have p = −b/(2a) and q = √|D|/(2a).

    Worked example

    Solve x² + x + 1 = 0.

    D = 1 − 4 = −3 < 0. Roots:

    x = (−1 ± i√3) / 2

    i.e. (−1 + i√3)/2 and (−1 − i√3)/2 — a conjugate pair, as expected.

    Equations of the form x² = −n

    x² + 4 = 0 ⇒ x² = −4 ⇒ x = ±2i.

    Sum and product of roots — unchanged

    If the roots of ax² + bx + c = 0 are α, β (real or complex), then

    • α + β = −b/a
    • α · β = c/a

    These Vieta formulas don’t care whether the roots are real or complex.

    13. Common pitfalls

    • √(−a)·√(−b) = √(ab) is wrong when both factors are negative. Convert to i first: √(−a)·√(−b) = (i√a)(i√b) = i² √(ab) = −√(ab).
    • Imaginary part is Im(z) = b, not ib. Easy to lose marks if you write Im(3 + 4i) = 4i.
    • Conjugate flips only the imaginary sign, not both signs. (3 + 4i)¯ = 3 − 4i, not −3 − 4i.
    • |z₁ + z₂| ≠ |z₁| + |z₂| in general — that’s the triangle inequality, not an equation. Equality holds only when the two complex numbers point in the same direction from the origin.
    • The principal argument lives in (−π, π]. A point in the third quadrant has negative principal argument, even though its “raw” angle measured anticlockwise is between π and 3π/2.
    • Don’t leave i in a denominator. Rationalise.
    • Powers of i: reduce mod 4, not mod 2. i⁵⁰ = i^(50 mod 4) = i² = −1.

    15. Looking ahead

    The next steps that build directly on this chapter:

    • De Moivre’s theorem. (cos θ + i sin θ)ⁿ = cos nθ + i sin nθ. It powers the formulas for nth roots of unity and trigonometric identity derivations.
    • Roots of unity. Solutions of zⁿ = 1 are n equally-spaced points on the unit circle, deeply connected to combinatorics and number theory.
    • Euler’s identity. e^(iθ) = cos θ + i sin θ, turning polar form into an exponential and unifying complex exponentiation with trigonometry. This shows up immediately in physics and engineering.
    • Polynomials. The Fundamental Theorem of Algebra says every non-constant polynomial with complex coefficients has at least one complex root — i.e. once we admit complex numbers, every polynomial factors completely. Class 11/12 don’t prove this but use it implicitly.
    • Loci in the Argand plane. Equations like |z − a| = r (a circle), |z − a| = |z − b| (a perpendicular bisector), and arg((z − a)/(z − b)) = α (an arc) translate algebraic conditions into geometric pictures, and are a staple of JEE problems.

    The thing to internalise from this chapter is that every algebraic manipulation you already know (factoring, quadratic formula, polynomial division) extends to complex numbers without modification — the only new rule is i² = −1.

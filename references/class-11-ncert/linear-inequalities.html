    Linear Inequalities (Class 11, Chapter 6)

    A written tutorial covering the NCERT Class 11 chapter on inequalities in one variable: terminology, the algebraic rules that govern manipulating an inequality, the standard solution procedure, representation on the number line and in interval notation, and worked NCERT-style problems.

    1. Equality vs inequality

    You already use the = sign comfortably. “I have 5 apples” becomes

    number of apples = 5

    Now flex the language slightly:

    • “I have at most 5 apples” — so the number could be 5 or any value below it. We write ≤ 5.
    • “I have at least 5 apples” — so the number is 5 or above. We write ≥ 5.
    • “I have fewer than 5 apples” — strictly below. < 5.
    • “I have more than 5 apples” — strictly above. > 5.

    These four symbols <, >, ≤, ≥ are called inequalities. A statement that uses one of them in place of = is called an inequation (sometimes also “an inequality”).

    Symbol Reads as Strict?
    < less than strict
    > greater than strict
    ≤ less than or equal to slack
    ≥ greater than or equal to slack
    • Strict inequalities: only < or >. The boundary value is not allowed.
    • Slack inequalities: ≤ or ≥. The boundary value is allowed.

    2. Classification of inequalities

    The same labels you used for equations (linear, quadratic, one-variable, two-variable) apply here, with “equation” replaced by “inequation”.

    Form Name
    ax + b < 0, ax + b > 0, etc. linear inequation in one variable
    ax + by < c, ax + by > c, etc. linear inequation in two variables
    ax² + bx + c < 0, etc. quadratic inequation in one variable

    This chapter focuses on the first row — solving linear inequalities in one variable and pictur­ising their solution sets.

    3. What counts as a “solution”?

    A real number x = k is a solution of an inequation if substituting k into the inequation makes the resulting statement true.

    Example. Take 2x < 200. Try x = 0: 2·0 = 0, and 0 < 200 is true → x = 0 is a solution. Try x = 200: 2·200 = 400, and 400 < 200 is false → x = 200 is not a solution.

    Unlike a linear equation, a linear inequation usually has infinitely many solutions. They form an interval on the number line.

    4. Algebra of inequalities — the four rules

    The cornerstone of the chapter. Two of these rules behave exactly like equations; two of them have a twist.

    Rule 1 — Adding the same number to both sides

    If a < b, then a + c < b + c   (for any real c)

    Signs do not change. Works for positive, negative, or zero c.

    Rule 2 — Subtracting the same number from both sides

    If a < b, then a − c < b − c   (for any real c)

    Signs do not change. Same as Rule 1 with c replaced by −c.

    Rule 3 — Multiplying or dividing by a positive number

    If a < b and c > 0, then a·c < b·c   and   a/c < b/c

    Signs do not change.

    Rule 4 — Multiplying or dividing by a negative number (the twist)

    If a < b and c < 0, then a·c > b·c   and   a/c > b/c

    The inequality sign flips. < becomes >, ≤ becomes ≥, and vice versa.

    Why the flip?

    A concrete check. Start with the true statement 3 < 5. Multiply both sides by −1:

    • Left side becomes −3.
    • Right side becomes −5.

    Is −3 < −5? No — −3 lies to the right of −5 on the number line, so −3 > −5. The inequality direction has reversed. Same thing happens with division.

    This single rule is the source of essentially every avoidable error in the chapter. Anytime a step involves multiplying or dividing by a negative number, mark it and flip the sign.

    Summary table

    Operation on both sides Effect on sign
    Add same number unchanged
    Subtract same number unchanged
    Multiply by positive number unchanged
    Divide by positive number unchanged
    Multiply by negative number flips
    Divide by negative number flips

    5. The standard solution procedure

    For a linear inequation in one variable, the workflow is essentially the same as for a linear equation, with rule 4 added in.

    1. Collect all x terms on one side, all constants on the other. Use add/subtract — no flip.
    2. Simplify both sides.
    3. Divide (or multiply) by the coefficient of x to isolate x. If that coefficient is negative, flip the inequality.
    4. Read the answer: x > k, x < k, x ≥ k, or x ≤ k.
    5. Express the solution set as required: integers, naturals, real numbers, an interval, or a graph on the number line.

    Reading-the-arrow tip

    After you finish, you’ll have something like x > −4 or −3 ≥ x. Always rewrite the answer so x sits on the left. The “mouth” of the inequality opens toward the larger quantity:

    • x > −4 reads “x is greater than −4”.
    • −3 ≥ x is the same as x ≤ −3.

    6. Representing solutions

    6.1 On the number line

    After you’ve solved for x, plot on a horizontal line.

    • Open circle ○ at the boundary if the inequality is strict (< or >). The boundary is not in the solution.
    • Filled circle ● at the boundary if the inequality is slack (≤ or ≥). The boundary is in the solution.
    • Arrow pointing toward the side that satisfies the inequality, extending to +∞ or −∞.

    Example: x > −4 looks like

                         ○─────────────►
                     |
   ──┼────┼────┼────┼────┼────┼────┼──
    −6   −5   −4   −3   −2   −1    0

    Example: x ≤ 2 looks like

       ◄─────────────●
                 |
   ──┼────┼────┼────┼────┼────┼──
    −2   −1    0    1    2    3

    6.2 In interval notation

    Inequality Interval
    x < a (−∞, a)
    x ≤ a (−∞, a]
    x > a (a, ∞)
    x ≥ a [a, ∞)
    a < x < b (a, b)
    a ≤ x ≤ b [a, b]

    A round bracket ( or ) means the endpoint is excluded. A square bracket [ or ] means the endpoint is included. Infinity always gets a round bracket — you cannot “reach” ∞.

    6.3 Solution set type matters

    What counts as a valid x depends on the problem’s universe:

    • Natural numbers (ℕ = {1, 2, 3, …}): pick from the inequality the values that are natural.
    • Integers (ℤ = {…, −2, −1, 0, 1, 2, …}): pick integer values.
    • Real numbers (ℝ): give the answer as an interval (or union of intervals).

    The same inequation can have wildly different “solution sets” depending on which of these is meant. Always check what the question asks for.

    7. Worked NCERT-style problems

    7.1 Two ways to read the same inequation

    Q. Solve 24x < 100 for (i) x is a natural number, (ii) x is an integer.

    Divide both sides by 24 (positive, so no flip):

    x < 100/24 = 25/6 ≈ 4.166…
      1. Natural numbers less than 25/6: {1, 2, 3, 4}.
      1. Integers less than 25/6: {…, −3, −2, −1, 0, 1, 2, 3, 4}.

    7.2 Dividing by a negative number

    Q. Solve −12x > 30 for (i) x is a natural number, (ii) x is an integer.

    Divide both sides by −12. Negative number, so flip:

    x < 30 / (−12) = −2.5
      1. Natural numbers less than −2.5: none (natural numbers are positive). The solution set is empty.
      1. Integers less than −2.5: {−3, −4, −5, …}.

    7.3 Constant on the wrong side

    Q. Solve 5x − 3 < 7 for (i) x is an integer, (ii) x is a real number.

    Step 1: Add 3 to both sides → 5x < 10. (No flip.) Step 2: Divide both sides by 5 → x < 2. (No flip.)

      1. Integers less than 2: {1, 0, −1, −2, …}.
      1. Real numbers less than 2: interval (−∞, 2).

    7.4 Same on the right side

    Q. Solve 3x + 8 > 2 for (i) x is an integer, (ii) x is a real number.

    3x + 8 > 2
3x > −6           (subtract 8)
x > −2            (divide by 3, positive — no flip)
      1. Integers greater than −2: {−1, 0, 1, 2, 3, …}.
      1. Real numbers greater than −2: interval (−2, ∞).

    7.5 Variables on both sides

    Q. Solve 4x + 3 < 5x + 7 and represent the solution on a number line (over the reals).

    4x + 3 < 5x + 7
3 < x + 7         (subtract 4x)
−4 < x            (subtract 7)
x > −4

    Solution set: (−4, ∞). Number-line representation: open circle at −4, arrow to the right.

    7.6 A flip in the middle

    Q. Solve 3x − 7 > 5x − 1 over the reals.

    3x − 7 > 5x − 1
−7 > 2x − 1          (subtract 3x)
−6 > 2x              (add 1)
−3 > x               (divide by 2 — positive, no flip)

    Rewrite with x on the left: x < −3. Solution interval (−∞, −3). Open circle at −3, arrow to the left.

    Notice we never divided by a negative number in this problem. We could have, by moving 5x to the left and −7 to the right — but that would have left a negative coefficient on x. Habit to develop: rearrange so the x-coefficient ends up positive. It saves you the cognitive load of remembering to flip.

    7.7 An inequality with a fraction

    Q. Solve (2x − 1)/3 ≥ (3x − 2)/4 − (2 − x)/5 and graph the solution.

    Combine the right side over the common denominator 20:

    (3x − 2)/4 − (2 − x)/5 = [5(3x − 2) − 4(2 − x)] / 20
                      = [15x − 10 − 8 + 4x] / 20
                      = (19x − 18) / 20

    So the inequality becomes

    (2x − 1)/3 ≥ (19x − 18)/20

    Cross-multiply by 60 (positive — no flip), or equivalently multiply both sides by 20 and 3:

    20(2x − 1) ≥ 3(19x − 18)
40x − 20 ≥ 57x − 54
54 − 20 ≥ 57x − 40x        (move x-terms right, constants left)
34 ≥ 17x
2 ≥ x

    So x ≤ 2. Solution interval (−∞, 2]. Number line: filled circle at 2, arrow to the left.

    7.8 Triple manipulation

    Q. Solve 3(1 − x) < 2(x + 4) over the reals.

    3 − 3x < 2x + 8        (expand)
3 − 8 < 2x + 3x        (move x-terms right, constants left)
−5 < 5x
−1 < x
x > −1

    Solution interval (−1, ∞). Open circle at −1, arrow right.

    7.9 When the coefficient becomes negative

    Q. Solve 5 − 2x ≤ 7.

    −2x ≤ 7 − 5
−2x ≤ 2
x ≥ −1               (divide by −2 — FLIP)

    Solution interval [−1, ∞). Filled circle at −1, arrow right.

    7.10 Compound inequality (introductory)

    Q. Solve −3 ≤ 4 − 7x/2 ≤ 18.

    Solve both pieces simultaneously by manipulating all three parts at once.

    −3 ≤ 4 − 7x/2 ≤ 18
−7 ≤ −7x/2 ≤ 14         (subtract 4 throughout)
−14 ≤ −7x ≤ 28          (multiply by 2 — positive, no flip)
2 ≥ x ≥ −4              (divide by −7 — FLIP both inequalities)
−4 ≤ x ≤ 2              (rewrite in standard order)

    Solution interval [−4, 2]. Filled circles at both endpoints, segment between them shaded.

    8. A note on equivalence

    Two inequations are equivalent if they have the same solution set. Each of Rules 1–4 produces an equivalent inequation, provided you remember to flip the sign in Rule 4.

    That’s the deep reason these rules are safe: every step preserves the solution set.

    If you ever multiply both sides by an expression that might be negative (e.g. (x − 1)), you’ve stopped working with a known sign and can no longer apply Rules 3/4 unconditionally. For linear inequalities this rarely comes up — you only divide by constants in this chapter — but keep it in mind for later.

    9. Common pitfalls

    • Forgetting to flip the sign when multiplying or dividing by a negative number. The single most common mistake.
    • Flipping the sign on addition or subtraction. Don’t — those operations never change the direction.
    • Treating < and ≤ interchangeably. They differ at the endpoint: open vs filled circle, round vs square bracket. Endpoint inclusion changes whole-marks-worth of answers in exams.
    • Using ( for ±∞ versus [. Infinity always gets a round bracket; it isn’t a number that can be included.
    • Mixing up “natural” and “integer” — natural numbers start at 1 (NCERT) and exclude 0 and negatives. If the question says “x is a natural number” and your interval is (−∞, 4), the answer is {1, 2, 3}, not {0, 1, 2, 3} and not {…, −1, 0, 1, 2, 3}.
    • Cross-multiplying without checking signs of denominators. With constant denominators (the only case in this chapter), this is safe; with variable denominators, you must split into cases.
    • Leaving x with a negative coefficient. You can — just remember to flip when dividing. But the habit of arranging things so x ends up positive on the left side prevents a lot of slip-ups.

    10. Quick reference card

    SYMBOLS
  <   strict less than           ≤   less than or equal to (slack)
  >   strict greater than        ≥   greater than or equal to (slack)

RULES (a < b   ⇒   ...)
  a + c < b + c                  for any real c
  a − c < b − c                  for any real c
  a·c < b·c    if c > 0          sign unchanged
  a/c < b/c    if c > 0          sign unchanged
  a·c > b·c    if c < 0          sign FLIPS
  a/c > b/c    if c < 0          sign FLIPS

INTERVALS
  x < a     ↔ (−∞, a)            x > a     ↔ (a, ∞)
  x ≤ a     ↔ (−∞, a]            x ≥ a     ↔ [a, ∞)
  a < x < b ↔ (a, b)             a ≤ x ≤ b ↔ [a, b]

GRAPH
  open circle ○   = boundary excluded   (strict)
  filled circle ● = boundary included   (slack)
  arrow points toward the side that satisfies the inequality

    Solution algorithm:

    1. Collect x-terms on one side, constants on the other (no flip).
2. Simplify.
3. Divide by coefficient of x.
     coefficient positive ⇒ no flip
     coefficient negative ⇒ FLIP the inequality
4. Read x's bound; rewrite with x on the left if needed.
5. Translate to the required form: integer list, natural list,
   interval, or graph.

    11. Looking ahead

    Linear inequalities in one variable open the door to several richer topics:

    • Linear inequalities in two variables. ax + by < c defines a half-plane in the xy-plane. The boundary is the line ax + by = c; the inequality picks the side. Strict vs slack again decides whether the boundary line is dashed or solid.
    • Systems of linear inequalities. Intersect several half-planes to get a feasible region — a polygon (possibly unbounded). This is the geometric heart of linear programming.
    • Linear programming. Optimise a linear objective Z = px + qy over a feasible region. Class 12 builds on this directly.
    • Quadratic and rational inequalities. x² − 5x + 6 ≤ 0 is solved by factoring and sign-charting; rational inequalities like (x − 1)/(x + 2) > 0 add the wrinkle that the denominator’s sign matters and that you cannot multiply across it without splitting cases.
    • Absolute-value inequalities. |x − 3| < 5 unpacks to −5 < x − 3 < 5, i.e. −2 < x < 8. The pattern |x − a| < r always describes an interval of radius r around a.
    • Inequalities in calculus and analysis. Continuity, limits, and the epsilon–delta definitions all live in the language of inequalities. The triangle inequality |a + b| ≤ |a| + |b| underlies a huge amount of higher mathematics.

    The single habit to internalise: whenever you multiply or divide an inequality by something, ask “is this positive or negative?” That one check, performed automatically, will keep you out of the most common trap in the entire chapter.

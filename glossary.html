Glossary

Plain-English definitions for every term named in this curriculum

This glossary is the canonical, plain-English reference for every term any lesson formally introduces in its ## Name beat. Each entry gives:

  • Plain-English one-liner. What the term means, in words you could say aloud to a friend.
  • First named in. The lesson where this term is first defined — your starting point if you want the full story, the worked examples, and the “where this lives in the world” connections.

This page grows with the curriculum. If a term is missing from a recently written lesson, that’s a bug — please open an issue.

How to use this page

You can land on this page two ways:

  1. Navbar → Glossary. Browse alphabetically, or use the table of contents on the right.
  2. Inline link from a lesson. Many lessons cross-link to specific entries. Clicking returns you to the same scroll position via your browser’s back button.

The glossary is not a substitute for the lessons. A definition tells you what a thing is; a lesson teaches you why it matters. Use the First named in link whenever you want the full picture.

A

Absolute extremum

The largest (max) or smallest (min) value of f on a closed interval [a, b]; found by comparing f at interior critical points and at the endpoints.

First named in: Class 12, Chapter 6 — Application of Derivatives (Maxima and Minima)

Action value

q_\pi(s, a) = \mathbb{E}_\pi[G_t \mid s_t = s, a_t = a] — expected return from taking a in s and following \pi thereafter. Enables model-free greedy improvement (you can pick \arg\max_a q without knowing the model).

First named in: MFRL — Lesson 08, From State Values to Action Values

Actor-critic

A class of RL algorithms with two parameterised functions: an actor (policy \pi_\theta) and a critic (value \hat v or \hat q). The critic is updated by TD; the actor is updated by policy gradient using the critic’s value estimate.

First named in: MFRL — Lesson 35, QAC: The Simplest Actor-Critic

Addition

Combining two numbers into a total — pictured on the number line as a hop to the right by the second number’s size.

First named in: Tier 1, Lesson 4 — Addition and Subtraction

Addition principle (counting)

If a task can be done by either method A (in m ways) or method B (in n ways) and the two methods are mutually exclusive, the total number of ways is m + n. The “OR” companion to the multiplication principle’s “AND”.

First named in: Class 11, Chapter 6 — Permutations and Combinations

Addition rule (probability)

P(A \cup B) = P(A) + P(B) - P(A \cap B). The general rule for the probability of a union; the overlap subtraction vanishes when the events are mutually exclusive.

First named in: Class 11, Chapter 14 — Probability

Adjoint of a matrix

The transpose of the cofactor matrix; satisfies A \cdot \text{adj}(A) = |A| \cdot I and gives the inverse via A^{-1} = \tfrac{1}{|A|}\text{adj}(A).

First named in: Class 12, Chapter 4 — Matrix Inverse and Cramer’s Rule

Advantage function

A_\pi(s, a) = q_\pi(s, a) - v_\pi(s) — how much better than the policy’s average action a is in state s. Used as a low-variance, unbiased substitute for q in policy-gradient updates.

First named in: MFRL — Lesson 36, A2C: Advantage Estimation

Algebraic connectivity

The second-smallest eigenvalue \lambda_2(L) of the graph Laplacian — small for graphs with bottlenecks, large for robustly connected ones (Fiedler 1973).

First named in: Tier 6, Lesson 14 — Spectral Graph Theory

Algebraic expression

A recipe built from numbers, variables, and operations, like 2x + 5 or (a + b)/2.

First named in: Tier 1, Lesson 11 — Variables

Algebraic fraction

An expression P(x)/Q(x) where P and Q are polynomials and Q is not the zero polynomial; the domain is all x for which Q(x) \ne 0. Manipulated by the same rules as numerical fractions: factor and cancel, common-denominator addition, multiply across, divide by reciprocal.

First named in: Kiselev Algebra Part I — Chapter 4, Algebraic fractions

Analytic function

A complex function f: D \subseteq \mathbb{C} \to \mathbb{C} whose derivative f'(z) exists as a complex limit independent of the direction of approach.

First named in: Tier 6, Lesson 5 — Complex Functions

Angle

A measure of the turn between two lines that meet at a common point. Measured in degrees: a full turn is 360°.

First named in: Tier 1, Lesson 12 — Shapes and Angles

Angle of depression

The angle between the horizontal at the observer and the line of sight downward to a lower object. Equal to the angle of elevation from the lower point (alternate-interior angles).

First named in: Class 10, Chapter 9 — Some Applications of Trigonometry

Angle of elevation

The angle between the horizontal at the observer and the line of sight upward to a higher object.

First named in: Class 10, Chapter 9 — Some Applications of Trigonometry

Antiderivative

A function whose derivative gives back the original function. If F'(x) = f(x), then F is an antiderivative of f — and so is F + C for any constant.

First named in: Tier 4, Lesson 9 — The Fundamental Theorem of Calculus

Area

The size of the surface inside a flat shape, measured in square units (m², cm²).

First named in: Tier 1, Lesson 13 — Area, Perimeter, Volume

Arithmetic-geometric progression

A sequence whose k-th term is the term-by-term product of an AP and a GP: T_k = (a + (k-1)d)\,r^{k-1}. Summed by the “multiply by r and subtract” trick, which converts it into a plain GP plus a boundary term.

First named in: Class 11, Chapter 8 — Sequences and Series (8b)

Arithmetic sequence

A sequence in which each term is the previous term plus a fixed amount, called the common difference. Example: 3, 7, 11, 15, \ldots

First named in: Tier 2, Lesson 15 — Sequences (Intro)

Asymptote

A line that a curve approaches arbitrarily closely without ever touching. The hyperbola x^2/a^2 - y^2/b^2 = 1 has asymptotes y = \pm (b/a) x.

First named in: Class 11, Chapter 10 — Conic Sections (Ellipse and Hyperbola)

Axiom (common notion)

A general logical principle accepted without proof, applicable beyond geometry — for example, “things equal to the same thing are equal to one another.”

First named in: Class 9, Chapter 5 — Introduction to Euclid’s Geometry

Axiomatic probability

Probability defined by Kolmogorov’s three axioms — non-negativity, P(S) = 1, and additivity for mutually exclusive events — rather than by counting equally-likely outcomes.

First named in: Class 11, Chapter 14 — Probability

B

Baird’s counterexample

A 7-state MDP with linear function approximation on which off-policy semi-gradient Q-learning’s parameters diverge to infinity. The canonical demonstration of the deadly triad (Baird 1995).

First named in: MFRL — Lesson 30, Q-Learning with FA and the Deadly Triad

Banach fixed-point theorem

Every contraction on a non-empty complete metric space has a unique fixed point, reached by iteration from any starting point at geometric rate L^n. The workhorse for proving uniqueness of solutions to fixed-point equations across analysis, ODEs, and RL.

First named in: MFRL — Lesson 11, The Banach Fixed-Point Theorem

Base

Two related meanings, both about the number that “sets the scale”:

  1. Base of a place-value system. The number on which a place-value system is built. Base 10 (everyday), base 2 (binary, used by computers), base 16 (hex, used by programmers), base 60 (Babylonian, still used for hours and minutes).

  2. Base of an exponent. In an expression like a^n, the number a that is being multiplied by itself.

First named in (place-value sense): Tier 1, Lesson 3 — Place Value First named in (exponent sense): Tier 2, Lesson 7 — Exponents and Powers

Baseline invariance

The lemma that \mathbb{E}_{a \sim \pi}[\nabla \log \pi(a \mid s) \cdot b(s)] = 0 for any function b(s) of state alone. Subtracting any state-only baseline from q_\pi inside the policy gradient leaves the expected gradient unchanged but can reduce variance.

First named in: MFRL — Lesson 36, A2C: Advantage Estimation

Basis

A linearly independent set of vectors whose span is the whole space; equivalently, a minimal spanning set or a maximal independent set.

First named in: Tier 5, Lesson 3 — Vector Spaces, Linear Independence, Basis

Bayes’ theorem

P(A \mid B) = P(B \mid A) \cdot P(A) / P(B) — the rule for flipping a conditional probability. Turns “what you can measure” into “what you actually want to know.”

First named in: Tier 3, Lesson 14 — Conditional Probability and Independence

Bellman equation

The recursion V(s) = \mathbb{E}[R(s,a) + \gamma V(s')] defining the value of a state as immediate reward plus discounted future value; for the optimal value function the expectation is replaced by a max over actions, and the resulting operator is a \gamma-contraction with a unique fixed point.

First named in: Tier 7, Lesson 6 — Reinforcement Learning Theory

Bellman optimality equation

v^*(s) = \max_a \sum_{s'} P(s' \mid s, a)[R(s, a, s') + \gamma v^*(s')]. The non-linear fixed-point equation whose unique solution is the optimal state value v^*.

First named in: MFRL — Lesson 10, The Bellman Optimality Equation (BOE)

Bijection

A function that is both one-one (injective) and onto (surjective) — a perfect pairing of inputs with outputs that has an inverse.

First named in: Class 12, Chapter 1 — Relations and Functions

Binomial distribution

The PMF P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} counting the number of successes in n independent trials, each with probability p. Mean is np.

First named in: Tier 3, Lesson 15 — Discrete Probability Distributions

Binomial theorem

The identity (a+b)^n = \sum_{r=0}^{n} \binom{n}{r} a^{n-r} b^r — every term of the expansion is a binomial coefficient times a single power product.

First named in: Tier 3, Lesson 12 — The Binomial Theorem

Bootstrapping

Substituting a current value-function estimate into a target instead of waiting for the realised return; the trick that turns the Bellman equation into an iterative learning rule.

First named in: MFRL — Lesson 23, TD(0)

Borrow

The bookkeeping step in column-subtraction: when a column doesn’t have enough to subtract from, you take ten from the column to its left.

First named in: Tier 1, Lesson 4 — Addition and Subtraction

Burnside’s lemma

The number of orbits of a group action equals the average number of points fixed by each group element. Despite the name, due to Cauchy and Frobenius.

First named in: Tier 6, Lesson 3 — Group Actions

C

Carry

The bookkeeping step in column-addition: when a column’s total exceeds 9, the overflow moves into the column to its left.

First named in: Tier 1, Lesson 4 — Addition and Subtraction

Cauchy-Riemann equations

The conditions u_x = v_y and u_y = -v_x on the real and imaginary parts f = u + iv that characterise complex differentiability.

First named in: Tier 6, Lesson 5 — Complex Functions

Cauchy sequence

A sequence whose terms become arbitrarily close to one another: for every \epsilon > 0 there exists N with |a_m - a_n| < \epsilon for all m, n \ge N.

First named in: Tier 5, Lesson 15 — Continuity and Differentiability — Rigorous Foundation

Cauchy’s theorem

For an analytic f in a simply-connected region, \oint_C f(z)\, dz = 0 for every closed contour C. The keystone of complex analysis.

First named in: Tier 6, Lesson 5 — Complex Functions

Cavalieri’s principle

If two solids lie between the same two parallel planes and every plane parallel to those slices both solids in regions of equal area, the two solids have equal volume. The discrete ancestor of integration by slicing.

First named in: Kiselev Geometry Book II — Chapter 2, Polyhedra

Central Limit Theorem

For i.i.d. variables with finite variance, the standardised sample mean converges in distribution to a standard normal — regardless of the original distribution.

First named in: Tier 5, Lesson 9 — Variance, Standard Deviation, and the Central Limit Theorem

Chain rule

(f(g(x)))' = f'(g(x)) \cdot g'(x) — derivative of a composition is “outer derivative times inner derivative.”

First named in: Tier 4, Lesson 4 — Differentiation Rules

First named in: MML — Lesson 12, Eigendecomposition and Diagonalisation

ELBO (Evidence Lower BOund)

For any latent-variable model and any distribution q(z): \log p(\boldsymbol{x}) \ge \mathbb{E}_q[\log p(\boldsymbol{x}, z)] - \mathbb{E}_q[\log q(z)]. The right-hand side is the ELBO, maximised at the true posterior. The foundation of EM and variational inference.

First named in: MML — Lesson 30, The EM Algorithm

EM algorithm (Expectation-Maximisation)

Iterative algorithm for MLE in latent-variable models: E-step computes posterior over latents given current parameters; M-step re-fits parameters given the posterior. Monotone improvement of log-likelihood; converges to local maximum.

First named in: MML — Lesson 30, The EM Algorithm

Empirical Risk Minimisation (ERM)

Minimise the average loss on the training data: \min \tfrac{1}{N} \sum_n L(y_n, f_{\boldsymbol{w}}(\boldsymbol{x}_n)). The universal recipe of supervised learning.

First named in: MML — Lesson 22, Data, Models, ERM

Exponential family

A parametric family with density p(x \mid \boldsymbol{\theta}) = h(x) \exp(\boldsymbol{\theta}^\top \boldsymbol{T}(x) - A(\boldsymbol{\theta})). Sufficient statistics live in \boldsymbol{T}; most common distributions (Gaussian, Beta, Gamma, Poisson, Bernoulli) belong to this family.

First named in: MML — Lesson 19, Conjugacy and the Exponential Family

Gaussian Mixture Model (GMM)

A density p(\boldsymbol{x}) = \sum_k \pi_k \mathcal{N}(\boldsymbol{x} \mid \boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k) — weighted sum of K Gaussians. Used for soft clustering and density estimation; fitted by EM.

First named in: MML — Lesson 29, Gaussian Mixture Models

Hessian

The matrix \boldsymbol{H}_{ij} = \partial^2 f / \partial x_i \partial x_j of second partial derivatives of a scalar function. Symmetric (under continuity). Its eigenvalues at a critical point classify min / max / saddle.

First named in: MML — Lesson 14, Differentiation and Gradients

Hinge loss

\max(0, 1 - y \cdot \mathrm{score}) — the loss minimised by soft-margin SVM. Zero past the margin; linearly penalising margin violations.

First named in: MML — Lesson 31, SVM Primal

Hyperplane

A codimension-1 affine subspace; in \mathbb{R}^n, the solution set of a single linear equation \boldsymbol{w}^\top \boldsymbol{x} = b. The decision boundary of every linear classifier.

First named in: MML — Lesson 06, Affine Spaces and Hyperplanes

Inner product

A symmetric, bilinear, positive-definite function \langle \boldsymbol{u}, \boldsymbol{v}\rangle on a vector space; induces a norm \sqrt{\langle \boldsymbol{u}, \boldsymbol{u}\rangle} and an angle.

First named in: MML — Lesson 07, Norms, Inner Products, Lengths, Angles

Jacobian

For \boldsymbol{g} : \mathbb{R}^n \to \mathbb{R}^m, the matrix \boldsymbol{J}_{ij} = \partial g_i / \partial x_j of partial derivatives. Generalises the derivative to vector-valued functions; multivariable chain rule = Jacobian product.

First named in: MML — Lesson 14, Differentiation and Gradients

KKT conditions

Karush–Kuhn–Tucker conditions: stationarity, primal feasibility, dual feasibility, complementary slackness. Necessary at any smooth optimum of a constrained problem (under regularity); sufficient for convex problems.

First named in: MML — Lesson 21, Constrained and Convex Optimisation

Kernel (Mercer kernel)

A symmetric function k(\boldsymbol{x}, \boldsymbol{x}') whose Gram matrix is positive semi-definite for every finite set of points; equivalent to an inner product in some feature space. The basis of the kernel trick.

First named in: MML — Lesson 32, Dual SVM and Kernels

Kernel trick

Replace inner products \boldsymbol{x}^\top \boldsymbol{x}' in an algorithm with a kernel k(\boldsymbol{x}, \boldsymbol{x}'); the algorithm operates as if in a (possibly infinite-dimensional) feature space, without ever materialising features.

First named in: MML — Lesson 32, Dual SVM and Kernels

Lagrange multiplier

The dual variable associated with a constraint in the Lagrangian \mathcal{L} = f - \lambda h; geometrically, the proportionality constant between \nabla f and \nabla h at a constrained extremum.

First named in: MML — Lesson 21, Constrained and Convex Optimisation

Latent variable

An unobserved random variable that, together with parameters, generates observed data. Examples: cluster assignment in a GMM, “true score” in factor analysis, z in probabilistic PCA / VAE.

First named in: MML — Lesson 28, PCA as a Latent-Variable Model

Linear independence

Vectors \boldsymbol{v}_1, \dots, \boldsymbol{v}_k are linearly independent if no non-trivial linear combination equals zero: \sum \alpha_i \boldsymbol{v}_i = \boldsymbol{0} forces \alpha_1 = \dots = \alpha_k = 0.

First named in: MML — Lesson 04, Linear Independence and Basis

Linear mapping

A function \Phi : V \to W satisfying \Phi(\alpha \boldsymbol{u} + \beta \boldsymbol{v}) = \alpha \Phi(\boldsymbol{u}) + \beta \Phi(\boldsymbol{v}). Once a basis is chosen on each side, \Phi is represented by a matrix.

First named in: MML — Lesson 05, Rank and Linear Mappings

Mahalanobis distance

\sqrt{(\boldsymbol{x} - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\boldsymbol{x} - \boldsymbol{\mu})} — distance scaled by the inverse covariance; “natural” distance in Gaussian space.

First named in: MML — Lesson 18, Summary Statistics and the Gaussian

Marginal likelihood (evidence)

p(\mathcal{D} \mid M) = \int p(\mathcal{D} \mid \theta, M) p(\theta \mid M) d\theta — the probability of the data under model M with parameters integrated out. Used for Bayesian model selection.

First named in: MML — Lesson 23, Directed Graphical Models and Model Selection

Maximum a Posteriori (MAP)

The mode of the posterior p(\theta \mid D) — i.e. the most probable parameter given the data. Equivalent to regularised MLE with -\log prior as the regulariser.

First named in: MML — Lesson 25, Bayesian Linear Regression

Multivariate Gaussian

\mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\Sigma}) in \mathbb{R}^D: density \propto \exp(-\tfrac{1}{2}(\boldsymbol{x} - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1}(\boldsymbol{x} - \boldsymbol{\mu})). Closed under affine maps, marginalisation, and conditioning.

First named in: MML — Lesson 18, Summary Statistics and the Gaussian

Norm

A length function \lVert \cdot \rVert on a vector space, satisfying positive-definiteness, homogeneity, and triangle inequality. Common: L1, L2, L∞.

First named in: MML — Lesson 07, Norms, Inner Products, Lengths, Angles

Normal equations

\boldsymbol{X}^\top \boldsymbol{X} \boldsymbol{w} = \boldsymbol{X}^\top \boldsymbol{y} — the linear system whose solution is the least-squares (MLE) regression weight.

First named in: MML — Lesson 24, Linear Regression MLE

Orthogonal complement

For a subspace U \subseteq V, the set U^\perp of vectors in V orthogonal to every element of U. Always a subspace; \dim U + \dim U^\perp = \dim V.

First named in: MML — Lesson 08, Orthonormal Basis and Orthogonal Complement

Orthogonal projection

The unique map from V to a subspace U that returns the closest point in U to the input; equivalent to “drop a perpendicular.” Least-squares regression is orthogonal projection onto the column space.

First named in: MML — Lesson 09, Orthogonal Projections and Rotations

Orthonormal basis

A basis of pairwise orthogonal unit vectors. Coordinates in such a basis are inner products with the basis vectors; sum of squared coordinates is squared norm (Parseval’s identity).

First named in: MML — Lesson 08, Orthonormal Basis and Orthogonal Complement

Posterior

p(\theta \mid \mathcal{D}) — the conditional distribution of parameters given the data; obtained via Bayes’ theorem from prior and likelihood.

First named in: MML — Lesson 17, Probability Spaces and Bayes

Principal Component Analysis (PCA)

A dimensionality-reduction technique: find the directions (eigenvectors of the data covariance) that maximise variance — equivalently, minimise reconstruction error in projection.

First named in: MML — Lesson 26, PCA Max-Variance and Projection-Error

Probabilistic PCA

Generative model with continuous latent z \sim \mathcal{N}(\boldsymbol{0}, \boldsymbol{I}) and observed \boldsymbol{x} = \boldsymbol{W} z + \boldsymbol{\mu} + \boldsymbol{\epsilon}. MLE recovers classical PCA; gives posterior + sampling for free.

First named in: MML — Lesson 28, PCA as a Latent-Variable Model

Probability space

A triple (\Omega, \mathcal{F}, P): sample space, \sigma-algebra of events, probability measure. The formal setting of probability theory.

First named in: MML — Lesson 17, Probability Spaces and Bayes

Prior

p(\theta) — the distribution over parameters before observing the data. Updated via Bayes’ theorem into a posterior.

First named in: MML — Lesson 17, Probability Spaces and Bayes

Pseudo-inverse

\boldsymbol{A}^+ — for a matrix without a true inverse (non-square or singular), the minimum-norm least-squares solver. Computed from SVD: invert non-zero singular values, transpose.

First named in: MML — Lesson 13, SVD and Low-Rank Approximation

Rank

The dimension of the column space (equivalently, the row space) of a matrix; the number of linearly independent columns. Invariant under change of basis.

First named in: MML — Lesson 05, Rank and Linear Mappings

Regularisation

A penalty \lambda \Omega(\boldsymbol{w}) added to the training loss to discourage extreme weight values; equivalent to a constraint on \Omega(\boldsymbol{w}). Ridge: \Omega = \lVert \cdot \rVert_2^2. Lasso: \Omega = \lVert \cdot \rVert_1.

First named in: MML — Lesson 22, Data, Models, ERM

Responsibility (GMM)

r_{nk} = p(z_n = k \mid \boldsymbol{x}_n) — the posterior probability that component k generated point \boldsymbol{x}_n in a Gaussian mixture model. Fractional / soft cluster assignment.

First named in: MML — Lesson 29, Gaussian Mixture Models

Ridge regression

Linear regression with L_2 penalty: \min \lVert \boldsymbol{X}\boldsymbol{w} - \boldsymbol{y}\rVert^2 + \lambda \lVert \boldsymbol{w}\rVert^2. Closed form (\boldsymbol{X}^\top \boldsymbol{X} + \lambda \boldsymbol{I})^{-1} \boldsymbol{X}^\top \boldsymbol{y} — MAP under a Gaussian prior on \boldsymbol{w}.

First named in: MML — Lesson 25, Bayesian Linear Regression

Rotation matrix

An orthogonal matrix with determinant +1; preserves lengths, angles, and orientation. In \mathbb{R}^2: \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}.

First named in: MML — Lesson 09, Orthogonal Projections and Rotations

Spectral theorem (MML view)

Every real symmetric matrix has an orthonormal eigenbasis; equivalently, \boldsymbol{A} = \boldsymbol{Q}\boldsymbol{\Lambda}\boldsymbol{Q}^\top with \boldsymbol{Q} orthogonal and \boldsymbol{\Lambda} diagonal real.

First named in: MML — Lesson 12, Eigendecomposition and Diagonalisation

Stochastic Gradient Descent (SGD)

Gradient descent where the gradient is estimated from a mini-batch rather than the whole dataset; unbiased, variance \propto 1/B. Cornerstone of large-scale ML training.

First named in: MML — Lesson 20, Gradient Descent, SGD, Momentum

Subspace

A non-empty subset of a vector space, closed under addition and scalar multiplication (equivalently: contains the origin and is closed under both operations).

First named in: MML — Lesson 03, Vector Spaces and Subspaces

Sufficient statistic

A function \boldsymbol{T}(x) that summarises all information in the data relevant to inferring the parameter; in the exponential family, \boldsymbol{T} appears in the density and the MLE depends on the data only through it.

First named in: MML — Lesson 19, Conjugacy and the Exponential Family

Support Vector Machine (SVM)

Max-margin classifier: find the hyperplane separating two classes with the largest margin to the closest training points. Trained as a quadratic program; in the dual, depends on data only through inner products — enabling the kernel trick.

First named in: MML — Lesson 31, SVM Primal

Support vector

A training point exactly on the SVM margin (or inside it, for soft-margin); only support vectors contribute to the decision boundary.

First named in: MML — Lesson 31, SVM Primal

Singular Value Decomposition (SVD)

Universal matrix factorisation \boldsymbol{A} = \boldsymbol{U}\boldsymbol{\Sigma}\boldsymbol{V}^\top: orthogonal × diagonal (non-negative) × orthogonal. Truncating to the top-k singular values gives the best rank-k approximation (Eckart–Young).

First named in: MML — Lesson 13, SVD and Low-Rank Approximation

Trace

The sum of the diagonal entries of a square matrix; equal to the sum of its eigenvalues. Invariant under change of basis and under cyclic permutation of products.

First named in: MML — Lesson 10, Determinant, Trace, Eigenvalues

Vector space

A set with addition and scalar multiplication satisfying eight standard axioms; the abstract setting of linear algebra.

First named in: MML — Lesson 03, Vector Spaces and Subspaces

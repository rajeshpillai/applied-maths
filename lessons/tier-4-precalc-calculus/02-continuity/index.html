02 · Continuity

When the function is unbroken — and what that buys us

Pre-flight check

  1. Have you done 01 · Limits?
  2. Comfortable with the limit notation \lim_{x \to a} f(x) = L?
  3. Can you sketch a simple piecewise function?

Touch

Pick a function. Some are continuous; some have breaks.

viewof contSel = Inputs.select(["f(x) = x² (continuous)", "f(x) = 1/x (break at 0)", "f(x) = step (jump at 0)", "f(x) = floor(x) (jumps at every integer)"], {value: "f(x) = x² (continuous)", label: "Function:"})
{
  const W = 720, H = 280, pad = 30;
  const xMin = -3, xMax = 3, yMin = -3, yMax = 5;
  const px = v => pad + ((v - xMin) / (xMax - xMin)) * (W - 2 * pad);
  const py = v => H - pad - ((v - yMin) / (yMax - yMin)) * (H - 2 * pad);
  const svg = d3.create("svg").attr("viewBox", `0 0 ${W} ${H}`)
    .attr("style", "width:100%; max-width:760px; background:#fafafa;");
  svg.append("line").attr("x1", pad).attr("x2", W - pad).attr("y1", py(0)).attr("y2", py(0)).attr("stroke","#1f2933");
  svg.append("line").attr("x1", px(0)).attr("x2", px(0)).attr("y1", pad).attr("y2", H - pad).attr("stroke","#1f2933");

  let segments = [[]], lastX = null, lastY = null;
  const fn = (x) => {
    if (contSel.startsWith("f(x) = x²")) return x * x;
    if (contSel.includes("1/x")) return Math.abs(x) < 0.01 ? null : 1/x;
    if (contSel.includes("step")) return x < 0 ? -1 : 1;
    if (contSel.includes("floor")) return Math.floor(x);
  };
  for (let x = xMin; x <= xMax; x += 0.02) {
    const y = fn(x);
    if (y === null || y < yMin - 1 || y > yMax + 1) { segments.push([]); continue; }
    if (lastY !== null && Math.abs(y - lastY) > 0.5 && Math.abs(x - lastX) < 0.05) { segments.push([]); }
    segments[segments.length - 1].push({x, y});
    lastX = x; lastY = y;
  }
  for (const seg of segments) {
    if (seg.length < 2) continue;
    const p = seg.map((p, i) => (i === 0 ? "M " : "L ") + `${px(p.x)},${py(p.y)}`).join(" ");
    svg.append("path").attr("d", p).attr("fill","none").attr("stroke","#c2185b").attr("stroke-width",2);
  }
  return svg.node();
}

A function is continuous at a if you can draw it through x = a without lifting your pen. Three conditions: f(a) exists; \lim_{x \to a} f(x) exists; they agree.

Wonder

  1. Polynomials are continuous everywhere. Sketch and check.
  2. f(x) = 1/x: continuous everywhere except at x = 0. Why?
  3. The “step” function jumps at 0. Even though both one-sided limits exist, why is it discontinuous?

Discover

Problem 1

For f to be continuous at a, three things must hold:

  1. f(a) is defined.
  2. \lim_{x \to a} f(x) exists.
  3. \lim_{x \to a} f(x) = f(a).

For each function, identify whether it’s continuous at the given point. If not, which condition fails?

  1. f(x) = x^2 at x = 3.
  2. f(x) = 1/x at x = 0.
  3. Piecewise: f(x) = x^2 for x \neq 1, f(1) = 5 — at x = 1.
Full solution
  1. Yes, continuous. f(3) = 9, limit is 9.
  2. No — f(0) undefined (condition 1 fails).
  3. No — f(1) = 5 but \lim_{x \to 1} x^2 = 1. Different (condition 3 fails). This is a removable discontinuity — could be patched by defining f(1) = 1.
The three failure modes: removable (point hole), jump (limits disagree left vs right), infinite (function blows up).

Problem 2

Is the function f(x) = \begin{cases} x^2 & x \le 1 \\ 2x & x > 1 \end{cases} continuous at x = 1?

Hint 1 — a nudge Compute the left limit, right limit, and f(1).
Full solution
  • f(1) = 1 (from the x \le 1 branch).
  • Left limit: \lim_{x \to 1^-} x^2 = 1.
  • Right limit: \lim_{x \to 1^+} 2x = 2.

Left and right disagree → limit doesn’t exist → not continuous at 1. Jump discontinuity.

Piecewise functions are continuous iff the pieces agree at the boundary.

Problem 3

Make f(x) = \begin{cases} \dfrac{x^2 - 1}{x - 1} & x \neq 1 \\ k & x = 1 \end{cases} continuous at x = 1. What value of k?

Hint 1 — a nudge \lim_{x \to 1} (x^2 - 1)/(x - 1) = 2 (Lesson 01 P2). Set k to that.
Full solution k = 2. This removes the discontinuity, hence “removable discontinuity.”

Problem 4

Intermediate Value Theorem (IVT). If f is continuous on [a, b] and N is between f(a) and f(b), then there exists c \in (a, b) with f(c) = N.

Use IVT to show that f(x) = x^3 - x - 1 has a root in [1, 2].

Hint 1 — a nudge f is continuous (polynomial). Compute f(1) and f(2). Is 0 between them?
Full solution

f(1) = 1 - 1 - 1 = -1. f(2) = 8 - 2 - 1 = 5. N = 0 lies between -1 and 5. IVT guarantees some c \in (1, 2) with f(c) = 0.

IVT is the foundation of root-finding algorithms. Bisection method: shrink the bracket by half each step until the root is located to desired precision. Used everywhere from physics to finance.

Problem 5

Where are these functions continuous?

  1. f(x) = \dfrac{1}{x^2 - 4}
  2. g(x) = \sqrt{x - 3}
  3. h(x) = \tan x
Full solution
  1. Everywhere except x = \pm 2 (denominator zero).
  2. For x \ge 3 only (\sqrt{} requires non-negative argument).
  3. Everywhere except x = \pi/2 + k\pi (denominator \cos x = 0).
Where to look for discontinuities: denominators (zeros), even roots (negative arguments), logs (non-positive arguments), piecewise boundaries.

The Intermediate Value Theorem sounds obvious, yet a fully rigorous proof had to wait for Bernard Bolzano (1817), who used it to prove that any polynomial of odd degree has a real root. Bolzano’s paper Rein analytischer Beweis — “purely analytic proof” — was a self-conscious revolt against the geometric hand-waving of earlier calculus, and it sat almost unread for fifty years before Weierstrass and others rediscovered the same ideas.

Continuous = pen never lifts. All the formal machinery — three conditions, one-sided limits, removable vs jump vs infinite — is just bookkeeping for that one mental picture. Wherever the pen lifts, something interesting is happening to the function.

Name

A function f is continuous at a if all three hold:

  1. f(a) is defined.
  2. \lim_{x \to a} f(x) exists.
  3. \lim_{x \to a} f(x) = f(a).

If f is continuous at every point in an interval, it is continuous on that interval.

Types of discontinuity:

  • Removable (point “hole”): could be patched by redefining f(a).
  • Jump (left and right limits disagree).
  • Infinite (function blows up to \pm \infty).

Continuous building blocks: polynomials, rational functions (except where denominator zero), \sin, \cos, e^x, \ln x (for x > 0), \sqrt{x} (for x \ge 0). Composing or summing continuous functions gives a continuous function.

Intermediate Value Theorem (IVT): continuous functions on [a, b] take every value between f(a) and f(b).

Extreme Value Theorem (EVT): continuous functions on a closed [a, b] attain a maximum and a minimum.

Continuous = unbroken pen-stroke; Three conditions: defined, limit exists, they agree; IVT and EVT are the two great theorems continuity earns you.

Apply

Worked example

Use IVT to argue that f(x) = \cos x - x has a root in [0, 1].

A competent solver:

  1. Continuity: \cos x and x are continuous; their difference is continuous.
  2. Endpoints: f(0) = 1 - 0 = 1. f(1) = \cos 1 - 1 \approx 0.540 - 1 = -0.460.
  3. Sign change: f(0) > 0, f(1) < 0. So 0 is between them.
  4. IVT applies: there exists c \in (0, 1) with f(c) = 0.

(Numerically, c \approx 0.739 — the Dottie number, fixed point of cosine.)

Practice — Warm

W1. Is f(x) = x^3 - 5x + 2 continuous everywhere?

Full solution Yes — polynomials are continuous everywhere.

W2. Where is f(x) = 1/(x - 3) discontinuous?

Full solution At x = 3 (denominator zero, infinite discontinuity).

W3. Is \sin x continuous?

Full solution Yes, everywhere.

Practice — Working

P1. Make f(x) = \begin{cases} \dfrac{x^2 - 9}{x - 3} & x \neq 3 \\ k & x = 3 \end{cases} continuous. What is k?

Full solution \lim_{x \to 3} (x^2 - 9)/(x - 3) = \lim (x + 3) = 6. So k = 6.

P2. Show x^4 - 3x + 1 = 0 has a root in [0, 1].

Hint 1 — a nudge f continuous. f(0) = 1, f(1) = -1. Sign change.
Full solution f(0) = 0 - 0 + 1 = 1 > 0. f(1) = 1 - 3 + 1 = -1 < 0. By IVT, exists c \in (0, 1) with f(c) = 0.

P3. For f(x) = \begin{cases} ax + 1 & x \le 2 \\ x^2 - 3 & x > 2 \end{cases}, find a so f is continuous everywhere.

Full solution At x = 2: left value 2a + 1; right limit 4 - 3 = 1. For continuity: 2a + 1 = 1 \Rightarrow a = 0.

Practice — Stretch

S1. Bisection method. Use IVT to find a root of x^3 - x - 1 = 0 to one decimal place. (Hint: f(1) < 0, f(2) > 0. Try midpoint 1.5; halve repeatedly.)

Full solution
  • f(1) = -1, f(2) = 5. Midpoint 1.5: f(1.5) = 3.375 - 1.5 - 1 = 0.875 > 0. Root in [1, 1.5].
  • f(1.25) \approx 1.953 - 1.25 - 1 = -0.297 < 0. Root in [1.25, 1.5].
  • f(1.375) \approx 2.6 - 1.375 - 1 = 0.225 > 0. Root in [1.25, 1.375].
  • f(1.31) \approx 2.249 - 1.31 - 1 = -0.061. Root in [1.31, 1.375].
  • …

Continuing: root \approx 1.3247 (the plastic number).

Bisection halves the bracket each step, gaining one binary digit per iteration. Slower than Newton’s method (Tier 4 later), but always converges if IVT applies.

S2. ReLU activation in neural networks. \text{ReLU}(x) = \max(0, x). Is it continuous? Differentiable (Lesson 03)?

Full solution

Continuous everywhere — left limit, right limit, value all agree at every point (including 0, where they all equal 0).

Differentiable everywhere except at 0 — at 0, the slope jumps from 0 (left) to 1 (right). The function has a “kink.” This makes ReLU “almost everywhere differentiable” — sufficient for ML gradient descent (in practice, the gradient at exactly 0 is taken to be 0 or 1 by convention).

ReLU’s continuity-but-not-differentiability is a feature — the sharp non-linearity helps neural networks express complex relationships. The whole world of deep learning was unlocked when researchers realised continuous-but-not-smooth activations work beautifully.

Common confusions

  1. “Continuous = smooth.” No — smooth means differentiable (Lesson 03). |x| is continuous everywhere but not differentiable at 0 (kink).
  2. “All functions in textbooks are continuous.” Many real-world functions have jumps: tax brackets, age groupings, postage rates, step responses in physics.
  3. “IVT gives the root location.” No — IVT only guarantees existence; finding the root requires algorithms (bisection, Newton’s method).

Takeaways

If you remember nothing else from this lesson, remember these:

  1. A function is continuous at a when “defined, has a limit, and they agree” — three boxes that all must tick.
  2. Discontinuities come in three flavours: removable (a patchable hole), jump (left and right disagree), and infinite (the function blows up).
  3. The Intermediate Value Theorem is what lets a sign change prove a root exists — the bedrock of every bracketed root-finder.

What this unlocks

  • 03 · Derivative — derivatives exist only where the function is “smooth” (a stronger condition).
  • Tier 5 — Real Analysis — careful \epsilon-\delta formulation; uniform continuity; etc.

Where this matters

AI / Machine Learning. ReLU and similar activations (S2) trade smoothness for expressivity. The choice between continuous-but-non- smooth and smooth activations shapes deep learning practice.

Numerical methods. Bisection method (S1), Newton’s method, and many root-finders rely on IVT-style guarantees.

Mathematical research. Continuity is the foundational property of functions in topology and analysis. The whole field of general topology generalises continuity to abstract spaces.

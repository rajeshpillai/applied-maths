Ch 7 — Coordinate Geometry

Pythagoras between two pairs of numbers, and a midpoint that needs no measurement

Pre-flight check

  1. Can you plot the point (3, -2) on graph paper?
  2. Can you state and use the Pythagorean theorem? See t2.pythagorean-theorem.
  3. Do you know what an “x-axis” and “y-axis” are? See cbse9.coordinate-geometry.
  4. Can you simplify \sqrt{50} to 5\sqrt{2}?

Three yeses, you are ready.

Touch

Two friends are at points A = (1, 2) and B = (5, 5) on a flat field. Use the widget — drag B — and watch how the distance between them changes.

viewof bx = Inputs.range([-5, 9], {value: 5, step: 1, label: "B's x"})
viewof by = Inputs.range([-5, 9], {value: 5, step: 1, label: "B's y"})
{
  const w = 380, h = 320, padX = 30, padY = 20;
  const xS = d3.scaleLinear().domain([-6, 10]).range([padX, w - padX]);
  const yS = d3.scaleLinear().domain([-6, 10]).range([h - padY, padY]);
  const A = [1, 2], B = [bx, by];
  const svg = d3.create("svg").attr("width", w).attr("height", h)
    .style("background", "#fafafa").style("border", "1px solid #ddd");
  // grid
  for (let i = -6; i <= 10; i++) {
    svg.append("line").attr("x1", xS(i)).attr("x2", xS(i))
      .attr("y1", yS(-6)).attr("y2", yS(10)).attr("stroke", "#eee");
    svg.append("line").attr("y1", yS(i)).attr("y2", yS(i))
      .attr("x1", xS(-6)).attr("x2", xS(10)).attr("stroke", "#eee");
  }
  svg.append("line").attr("x1", xS(-6)).attr("x2", xS(10))
    .attr("y1", yS(0)).attr("y2", yS(0)).attr("stroke", "#888");
  svg.append("line").attr("y1", yS(-6)).attr("y2", yS(10))
    .attr("x1", xS(0)).attr("x2", xS(0)).attr("stroke", "#888");
  // right triangle
  svg.append("line").attr("x1", xS(A[0])).attr("y1", yS(A[1]))
    .attr("x2", xS(B[0])).attr("y2", yS(A[1])).attr("stroke", "#26a")
    .attr("stroke-dasharray", "3,3");
  svg.append("line").attr("x1", xS(B[0])).attr("y1", yS(A[1]))
    .attr("x2", xS(B[0])).attr("y2", yS(B[1])).attr("stroke", "#26a")
    .attr("stroke-dasharray", "3,3");
  svg.append("line").attr("x1", xS(A[0])).attr("y1", yS(A[1]))
    .attr("x2", xS(B[0])).attr("y2", yS(B[1])).attr("stroke", "#c33")
    .attr("stroke-width", 2);
  // points
  for (const [P, name] of [[A, "A"], [B, "B"]]) {
    svg.append("circle").attr("cx", xS(P[0])).attr("cy", yS(P[1]))
      .attr("r", 4).attr("fill", "#333");
    svg.append("text").attr("x", xS(P[0]) + 6).attr("y", yS(P[1]) - 6)
      .attr("font-size", 12).text(`${name}(${P[0]}, ${P[1]})`);
  }
  const dx = B[0] - A[0], dy = B[1] - A[1];
  const dist = Math.sqrt(dx * dx + dy * dy);
  svg.append("text").attr("x", padX).attr("y", h - 6).attr("font-size", 13)
    .text(`Δx = ${dx},  Δy = ${dy},  distance = √(${dx*dx} + ${dy*dy}) = ${dist.toFixed(3)}`);
  return svg.node();
}

The dashed lines show the horizontal and vertical “legs” of a right triangle whose hypotenuse is AB. The horizontal leg has length |x_2 - x_1|, the vertical leg |y_2 - y_1|.

Wonder

  1. The widget draws a right triangle every time. Why?
  2. If you had a way to compute the third side of a right triangle from the other two — would that give you the distance?
  3. Halfway between A and B — what point is that, exactly? Try a few placements of B.
  4. Could you compute the distance if both points had the same x-coordinate (a vertical line)?
TipWhat you might notice
  1. Yes — Pythagoras is the whole game. (3) The midpoint coordinates are the averages of the endpoint coordinates. (4) Yes — distance is just |y_2 - y_1|.

Discover

Problem 1 — Distance via Pythagoras

Find the distance between A(1, 2) and B(4, 6).

Hint 1 — a nudge Horizontal leg = 4 - 1 = 3. Vertical leg = 6 - 2 = 4.
Full solution d = \sqrt{3^2 + 4^2} = \sqrt{25} = 5.

Problem 2 — General formula

Derive a formula for the distance between P(x_1, y_1) and Q(x_2, y_2).

Hint 1 — a nudge Horizontal leg = x_2 - x_1 (or its absolute value). Vertical leg = y_2 - y_1. Apply Pythagoras.
Full solution d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.

Problem 3 — Midpoint

Find the midpoint of A(2, 3) and B(8, 11).

Hint 1 — a nudge The midpoint splits both legs in half. So both coordinates are averages.
Full solution M = \left(\dfrac{2 + 8}{2}, \dfrac{3 + 11}{2}\right) = (5, 7).

Problem 4 — Section formula (1:2)

A point P divides the segment AB in the ratio 1 : 2 (so AP : PB = 1 : 2). With A = (3, 4), B = (9, 16), find P.

Hint 1 — a nudge P is one-third of the way from A to B (because AP = \frac{1}{3}AB).
Hint 2 — the technique P_x = A_x + \frac{1}{3}(B_x - A_x) = \frac{2A_x + B_x}{3}? Check: with ratio m : n = 1 : 2, P = \dfrac{m B + n A}{m + n}.
Full solution P_x = \dfrac{1 \cdot 9 + 2 \cdot 3}{1 + 2} = \dfrac{9 + 6}{3} = 5. P_y = \dfrac{1 \cdot 16 + 2 \cdot 4}{3} = \dfrac{24}{3} = 8. P = (5, 8). Verify: AP = \sqrt{(5-3)^2 + (8-4)^2} = \sqrt{20}; PB = \sqrt{16 + 64} = \sqrt{80} = 2\sqrt{20}. Ratio = 1 : 2. ✓

Problem 5 — Section formula (general)

For any A(x_1, y_1), B(x_2, y_2) and ratio m : n, derive the formula for the dividing point.

Hint 1 — a nudge Mimic Problem 4. The dividing point splits each leg of the right triangle in the same ratio.
Full solution P = \left(\dfrac{m x_2 + n x_1}{m + n}, \dfrac{m y_2 + n y_1}{m + n} \right). The midpoint formula is the special case m = n = 1.

Problem 6 — Collinearity check

Check whether A(1, 1), B(4, 4), C(7, 7) are collinear.

Hint 1 — a nudge If collinear, AB + BC = AC (one of the three pairs).
Full solution AB = \sqrt{9 + 9} = 3\sqrt{2}. BC = 3\sqrt{2}. AC = 6\sqrt{2}. AB + BC = 6\sqrt{2} = AC. Yes — collinear.

The picture you have been working with — a plane labelled by two perpendicular axes — was published by René Descartes in La Géométrie (1637, an appendix to his Discours de la méthode). The often-told origin story has Descartes lying ill in bed, watching a fly walk across the ceiling, and realising he could pin down its position by two distances from two adjacent walls. True or apocryphal, the move was revolutionary: every geometric question (intersection, distance, midpoint) became an algebraic question, and every algebraic equation became a picture. Three and a half centuries later, that same dictionary is what lets a GPU translate the matrix algebra of a 3-D model into the pixels you see on screen.

The distance and section formulas are not new theorems — they are the Pythagorean theorem and similar triangles, re-spoken in coordinates. Once you internalise that, you stop memorising and start re-deriving on the fly: drop a perpendicular, read off the legs |x_2 - x_1| and |y_2 - y_1|, and Pythagoras finishes the job.

Name

Distance formula. The distance between P(x_1, y_1) and Q(x_2, y_2) is PQ = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.

Section formula. The point P that divides the segment AB in the ratio m : n (with A = (x_1, y_1), B = (x_2, y_2), and P between A and B — internal division) is P = \left(\dfrac{m x_2 + n x_1}{m + n}, \dfrac{m y_2 + n y_1}{m + n} \right).

Midpoint is the special case m = n = 1: M = \left(\dfrac{x_1 + x_2}{2}, \dfrac{y_1 + y_2}{2}\right).

A verse:

Subtract, then square, then square, then add — under the root they go. Distance falls out, true and clean — Pythagoras on show. Cut a segment m to n? Cross-multiply each end — m times far plus n times near, divide by m plus n.

Apply

Worked example

Find the coordinates of the point P that divides the segment from A(-1, 7) to B(4, -3) in the ratio 2 : 3.

m = 2, n = 3. P_x = \dfrac{2(4) + 3(-1)}{5} = \dfrac{8 - 3}{5} = 1. P_y = \dfrac{2(-3) + 3(7)}{5} = \dfrac{-6 + 21}{5} = 3.

P = (1, 3).

Practice — Warm

W1. Distance between (3, 0) and (0, 4).

Full solution \sqrt{9 + 16} = 5.

W2. Midpoint of (-2, 5) and (8, -1).

Full solution \left(\dfrac{-2 + 8}{2}, \dfrac{5 - 1}{2}\right) = (3, 2).

W3. Distance between origin and (7, -24).

Full solution \sqrt{49 + 576} = \sqrt{625} = 25.

Practice — Working

P1. Show that the points A(1, 7), B(4, 2), C(-1, -1), D(-4, 4) are the vertices of a square.

Hint 1 — a nudge Compute all four sides and the two diagonals.
Full solution AB = \sqrt{9 + 25} = \sqrt{34}. BC = \sqrt{25 + 9} = \sqrt{34}. CD = \sqrt{9 + 25} = \sqrt{34}. DA = \sqrt{25 + 9} = \sqrt{34}. All sides equal. Diagonals: AC = \sqrt{4 + 64} = \sqrt{68}. BD = \sqrt{64 + 4} = \sqrt{68}. Equal. So it’s a rhombus with equal diagonals — a square. ✓

P2. Find k so that (k, 2) is equidistant from (3, 4) and (-3, 7).

Hint 1 — a nudge d_1^2 = d_2^2.
Full solution (k - 3)^2 + (2 - 4)^2 = (k + 3)^2 + (2 - 7)^2 \Rightarrow k^2 - 6k + 9 + 4 = k^2 + 6k + 9 + 25 \Rightarrow -12k = 21 \Rightarrow k = -\dfrac{7}{4}.

P3. Find the ratio in which the point (-3, p) divides the segment from A(-5, -4) to B(-2, 3). Hence find p.

Hint 1 — a nudge Use section formula on the x-coordinate.
Full solution Let ratio m : n. -3 = \dfrac{-2m - 5n}{m + n} \Rightarrow -3m - 3n = -2m - 5n \Rightarrow m = 2n. Ratio 2 : 1. p = \dfrac{2(3) + 1(-4)}{3} = \dfrac{2}{3}.

Practice — Stretch

S1. Find the area of the triangle with vertices A(2, 3), B(-1, 0), C(2, -4) using the formula \text{Area} = \frac{1}{2}|x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)|.

Hint 1 — a nudge Plug in carefully. Watch signs.
Full solution

\frac{1}{2}|2(0 - (-4)) + (-1)((-4) - 3) + 2(3 - 0)| = \frac{1}{2}|2 \cdot 4 + (-1)(-7) + 2 \cdot 3| = \frac{1}{2}|8 + 7 + 6| = \frac{21}{2} sq units.

(Note: the area-of-triangle formula was retained in the rationalised syllabus despite some early confusion. It’s a standard 3-marker.)

S2. Find the centroid of the triangle with vertices A(1, 4), B(-2, -1), C(7, 3).

Hint 1 — a nudge The centroid is at the average of the three vertex coordinates.
Full solution G = \left(\dfrac{1 - 2 + 7}{3}, \dfrac{4 - 1 + 3}{3}\right) = (2, 2).

Board-Pattern Practice

Q1 (1 mark — MCQ). Distance between (0, 0) and (a \cos\theta, a \sin\theta) is: (a) a (b) a^2 (c) \sqrt{a} (d) None.

Full solution \sqrt{a^2 \cos^2\theta + a^2 \sin^2\theta} = \sqrt{a^2(\cos^2\theta + \sin^2\theta)} = \sqrt{a^2} = |a|. (a) a (assuming a > 0).

Q2 (1 mark — fill-in). The midpoint of the segment joining (-3, 5) and (7, -1) is ____.

Full solution \left(\dfrac{-3 + 7}{2}, \dfrac{5 - 1}{2}\right) = (2, 2).

Q3 (2 marks). Find the value of y for which the distance between the points P(2, -3) and Q(10, y) is 10 units.

Hint 1 — a nudge \sqrt{64 + (y + 3)^2} = 10.
Full solution (y + 3)^2 = 36 \Rightarrow y + 3 = \pm 6 \Rightarrow y = 3 or y = -9.

Q4 (2 marks). If the point P(x, y) is equidistant from A(5, 1) and B(-1, 5), prove that 3x = 2y.

Hint 1 — a nudge PA = PB \Rightarrow PA^2 = PB^2. Expand.
Full solution (x - 5)^2 + (y - 1)^2 = (x + 1)^2 + (y - 5)^2. Expand: x^2 - 10x + 25 + y^2 - 2y + 1 = x^2 + 2x + 1 + y^2 - 10y + 25. Simplify: -10x - 2y = 2x - 10y \Rightarrow 8y = 12x \Rightarrow 3x = 2y. \square

Q5 (3 marks). Find the ratio in which the line segment joining A(-2, -3) and B(3, 7) is divided by the y-axis. Find the y-coordinate of the dividing point.

Hint 1 — a nudge The y-axis has x = 0. Use the section formula’s x-coordinate.
Full solution 0 = \dfrac{3m - 2n}{m + n} \Rightarrow 3m = 2n \Rightarrow m : n = 2 : 3. y = \dfrac{2 \cdot 7 + 3 \cdot (-3)}{5} = \dfrac{14 - 9}{5} = 1. Ratio 2 : 3, dividing point (0, 1).

Q6 (3 marks). Show that the points A(7, 10), B(-2, 5), C(3, -4) are the vertices of an isosceles right triangle.

Hint 1 — a nudge Compute the three sides. Then check Pythagoras.
Full solution AB = \sqrt{81 + 25} = \sqrt{106}. BC = \sqrt{25 + 81} = \sqrt{106}. AC = \sqrt{16 + 196} = \sqrt{212}. AB = BC — isosceles. AB^2 + BC^2 = 106 + 106 = 212 = AC^2 — right angle at B. Isosceles right triangle. \square

Q7 (5 marks — long answer). The line segment joining A(2, -2) and B(3, 7) is divided by a point P such that \dfrac{AP}{PB} = \dfrac{2} {3}. Find: (a) the coordinates of P; (b) whether P lies on the line 2x + y - 4 = 0.

Hint 1 — a nudge Section formula with m : n = 2 : 3. Then plug into the line equation.
Full solution

(a) P_x = \dfrac{2(3) + 3(2)}{5} = \dfrac{12}{5}, P_y = \dfrac{2(7) + 3(-2)}{5} = \dfrac{8}{5}. P = \left(\dfrac{12}{5}, \dfrac{8}{5}\right).

(b) Check: 2 \cdot \dfrac{12}{5} + \dfrac{8}{5} - 4 = \dfrac{24}{5} + \dfrac{8}{5} - \dfrac{20}{5} = \dfrac{12}{5} \ne 0. So P does not lie on the line.

Takeaways

If you remember nothing else from this lesson, remember these:

  1. The coordinate plane turns geometry into algebra. Every point is an ordered pair (x, y); every line is an equation; every distance is Pythagoras.
  2. Distance formula is just Pythagoras with |x_2-x_1| and |y_2-y_1| as legs — and the section formula is just similar triangles writing themselves as a weighted average of coordinates.
  3. Three points are collinear iff one of the three pairwise distances equals the sum of the other two (or, equivalently, the triangle they form has area zero).

Common confusions

  1. “d = (x_2 - x_1) + (y_2 - y_1).” No — that gives a signed sum, not a length. Square, add, then root. (Even more dangerous: forgetting to square — losing the negative-sign safety.)
  2. “Section formula: \dfrac{n x_2 + m x_1}{m + n} vs \dfrac{m x_2 + n x_1}{m + n}.” It’s \dfrac{m x_2 + n x_1}{m + n} — the coefficients cross over. Memorise: with ratio m : n, the point leans towards B when m is larger.
  3. “The y-axis has y = 0.” No — the y-axis has x = 0 (every point on it has zero x-coordinate). The x-axis has y = 0.

Where this matters

Game development. Every collision check (“did the bullet hit the target?”) is a distance comparison in 2D or 3D. Every “spawn the enemy halfway between two waypoints” is a midpoint computation.

AI/ML. k-nearest-neighbour, k-means clustering, and almost every geometric ML algorithm rest on the distance formula generalised to n dimensions. Same Pythagoras, more terms under the root.

Research. Coordinate geometry — René Descartes, 1637 — was the breakthrough that made geometry algebraic. Almost every modern mathematical structure (manifolds, vector spaces, fibre bundles) is a descendant of this idea: put coordinates on geometry.

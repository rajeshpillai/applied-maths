11 · The Banach Fixed-Point Theorem

Self-contained presentation of the contraction mapping theorem — the workhorse for proving BOE has a unique solution.

Pre-flight check

Be honest. Are you ready for this lesson?

  1. Can you state what a Cauchy sequence is, and why “Cauchy” is not the same as “convergent” in a general metric space?
  2. Do you know what completeness of a metric space means? (Hint: every Cauchy sequence has a limit inside the space.)
  3. Can you compute distances |x - y| on the real line and draw a sequence that gets closer and closer to \sqrt{2}?

If any of these felt fuzzy, revisit Tier 5 — Real-Analysis Foundation before you start.

Touch

Before any vocabulary, you are going to watch a sequence converge. Type a starting number x_0 > 0, and the widget repeats one rule:

x_{n+1} = \tfrac{1}{2}\!\left(x_n + \tfrac{2}{x_n}\right).

viewof x0 = Inputs.range([0.1, 10], {value: 1, step: 0.1, label: "Starting x₀"})
viewof steps = Inputs.range([1, 12], {value: 6, step: 1, label: "How many steps?"})
{
  const f = x => 0.5 * (x + 2 / x);
  let x = x0;
  const rows = [["n", "xₙ", "|xₙ − √2|"]];
  for (let n = 0; n <= steps; n++) {
    rows.push([n, x.toFixed(10), Math.abs(x - Math.SQRT2).toExponential(2)]);
    x = f(x);
  }
  return html`<div style="font-family: monospace; font-size: 0.95rem;">
    ${rows.map(r => html`<div>${r[0].toString().padEnd(4)} | ${r[1].toString().padEnd(14)} | ${r[2]}</div>`)}
  </div>`;
}

Try x_0 = 1. Try x_0 = 5. Try x_0 = 0.1. The sequence rushes to the same number — and the error in the right column drops by something like a factor of two on every line, and then much faster.

Don’t worry yet about why. Just notice: no matter where you start, the process settles down at one point.

Wonder

Sit with these — no answers yet.

  1. The function f(x) = \tfrac{1}{2}(x + 2/x) doesn’t look special. What is it about f that drags every starting point to the same place?
  2. If f pulled distances closer together by a fixed factor — say, every pair (x, y) becomes a pair half as far apart after applying f — would the sequence have to converge?
  3. The point everything settles at — call it x^* — must satisfy f(x^*) = x^*. Why? And could there be two such points?

Discover

Five short problems. Each one is a step in the proof of one big theorem. Don’t peek at Name until you’ve worked through all five.

Problem 1 — Define “shrinks distances”

Suppose f : \mathbb{R} \to \mathbb{R} has the property that for every pair x, y:

|f(x) - f(y)| \le L \,|x - y|, \quad \text{some fixed } 0 \le L < 1.

Show: if you apply f to two different starting points x_0, y_0, then after n steps the distance between them is at most L^n |x_0 - y_0|.

Hint 1 — a nudge Compute |f(x_0) - f(y_0)|, then |f(f(x_0)) - f(f(y_0))|. What’s the pattern?
Hint 2 — the technique Apply the bound once, then again, then again. Each application contributes one factor of L.
Full solution Let x_n = f^{(n)}(x_0) and y_n = f^{(n)}(y_0). We are given |x_1 - y_1| \le L\,|x_0 - y_0|. Inductively, suppose |x_n - y_n| \le L^n |x_0 - y_0|. Then |x_{n+1} - y_{n+1}| = |f(x_n) - f(y_n)| \le L\,|x_n - y_n| \le L^{n+1} |x_0 - y_0|. Since L < 1, L^n \to 0, so any two trajectories merge.

Problem 2 — At most one fixed point

Suppose x^* and y^* are both fixed points: f(x^*) = x^* and f(y^*) = y^*. Show x^* = y^*.

Hint 1 — a nudge Apply the contraction bound to the pair (x^*, y^*).
Full solution |x^* - y^*| = |f(x^*) - f(y^*)| \le L\,|x^* - y^*|. So (1 - L)|x^* - y^*| \le 0. Since L < 1, 1 - L > 0, forcing |x^* - y^*| = 0. A contraction has at most one fixed point.

Problem 3 — The orbit is Cauchy

Pick any starting x_0 and write x_n = f^{(n)}(x_0). Show that the successive distances decay geometrically: |x_{n+1} - x_n| \le L^n |x_1 - x_0|. Then conclude that for m > n, |x_m - x_n| \le |x_1 - x_0| \cdot \frac{L^n}{1 - L}.

Hint 1 — a nudge For the first part, apply Problem 1 with y_0 = x_1.
Hint 2 — the technique For the second part, write x_m - x_n = (x_m - x_{m-1}) + \cdots + (x_{n+1} - x_n), take absolute values, then sum a geometric series.
Full solution

Successive distances. |x_{n+1} - x_n| = |f(x_n) - f(x_{n-1})| \le L\,|x_n - x_{n-1}|. By induction, |x_{n+1} - x_n| \le L^n |x_1 - x_0|.

Telescoping. For m > n, |x_m - x_n| \le \sum_{k=n}^{m-1} |x_{k+1} - x_k| \le |x_1 - x_0| \sum_{k=n}^{m-1} L^k \le |x_1 - x_0| \cdot \frac{L^n}{1 - L}. The right-hand side \to 0 as n \to \infty, so (x_n) is a Cauchy sequence.

Problem 4 — Use completeness to land somewhere

You just showed (x_n) is Cauchy. Why does that mean the sequence has a limit? What property of \mathbb{R} are you secretly using?

Hint 1 — a nudge Look up the prerequisite lesson on Cauchy sequences and completeness.
Full solution

\mathbb{R} is complete: every Cauchy sequence converges to a point in \mathbb{R}. So x_n \to x^* for some x^* \in \mathbb{R}.

If we worked over \mathbb{Q} instead, the same Cauchy sequence might not have a limit in \mathbb{Q} — Newton iteration starting at x_0 = 1 is all rationals but converges to the irrational \sqrt{2}. Completeness is exactly the property that lets us conclude “the limit exists.”

Problem 5 — The limit is a fixed point, and the rate

Show that the limit x^* from Problem 4 satisfies f(x^*) = x^*. Then bound |x_n - x^*| in terms of L^n.

Hint 1 — a nudge For the fixed-point property, take the limit on both sides of x_{n+1} = f(x_n). You need f to be continuous — but every contraction is continuous (Problem 1 with L < 1 implies it’s even Lipschitz).
Hint 2 — the technique For the rate, let m \to \infty in the bound from Problem 3.
Full solution

Fixed point. f is continuous (Lipschitz with constant L). Take limits in x_{n+1} = f(x_n): x^* = \lim x_{n+1} = \lim f(x_n) = f(\lim x_n) = f(x^*).

Rate. Letting m \to \infty in |x_m - x_n| \le |x_1 - x_0| L^n / (1-L) gives |x^* - x_n| \le \frac{L^n}{1 - L}\,|x_1 - x_0|. The error shrinks at least geometrically. (For Newton’s iteration on \sqrt{2} it actually shrinks much faster — that’s a bonus, not the rule.)

A contraction is “shrinks pairs of points by a fixed factor L < 1.” The geometric shrinkage drives the orbit to a Cauchy sequence; completeness hands you the limit; the limit is forced to be the unique fixed point. That’s the entire mechanism — five lines, each tight.

Name

A map f : X \to X on a metric space (X, d) is a contraction mapping if there exists L \in [0, 1) with d(f(x), f(y)) \le L \, d(x, y) \quad \text{for all } x, y \in X.

Banach Fixed-Point Theorem. Let (X, d) be a non-empty complete metric space and f a contraction with constant L. Then:

  1. f has a unique fixed point x^* \in X.
  2. For any starting point x_0 \in X, the iteration x_{n+1} = f(x_n) converges to x^*.
  3. The convergence is geometric: d(x_n, x^*) \le \frac{L^n}{1-L}\,d(x_1, x_0).

A verse to remember it by:

Pull every pair a little closer; Iterate from anywhere you please. The orbit’s a Cauchy chain — completeness catches it at one fixed point.

Apply

Worked example — Newton iteration to \sqrt{2}

Let f(x) = \tfrac{1}{2}(x + 2/x) on X = [1, 2] with the usual distance.

Is f a contraction on X? Compute the derivative: f'(x) = \tfrac{1}{2}(1 - 2/x^2). On [1, 2], 2/x^2 \in [1/2, 2], so f'(x) \in [-1/2, 1/4]. Hence |f'(x)| \le 1/2, and by the Mean Value Theorem |f(x) - f(y)| \le \tfrac{1}{2}|x - y|. Take L = 1/2.

Fixed point. Solving x = \tfrac{1}{2}(x + 2/x) gives x^2 = 2, so x^* = \sqrt{2} \approx 1.41421.

Convergence rate (worst case). From x_0 = 1 we get x_1 = 1.5, so |x_1 - x_0| = 0.5. The Banach bound predicts |x_n - \sqrt{2}| \le \tfrac{(1/2)^n}{1 - 1/2} \cdot 0.5 = (1/2)^n.

The widget shows the actual error drops much faster — Newton’s iteration is locally quadratic. The Banach bound is a worst case; the actual rate can beat it. Either way, convergence is guaranteed.

Practice — Warm

W1. Show that f(x) = \cos(x) is a contraction on [0, 1] with L = \sin(1) \approx 0.841. (Find L from |f'|.)

Hint 1 — a nudge |f'(x)| = |\sin(x)|. Maximise on [0, 1].
Hint 2 — the technique |\sin(x)| is increasing on [0, 1], so the maximum is at x = 1.
Hint 3 — the first step Mean Value Theorem says |f(x) - f(y)| = |f'(c)|\,|x-y| for some c between x and y.
Full solution |\cos(x) - \cos(y)| = |-\sin(c)|\,|x - y| \le \sin(1)|x - y| for c \in [0,1]. Since \sin(1) < 1, f is a contraction. The fixed point is the famous Dottie number \approx 0.7390851, the unique solution of \cos(x) = x.

W2. Iterate f(x) = \cos(x) from x_0 = 0.5 for five steps. Compare the actual error to the Banach worst-case bound.

Hint 1 — a nudge |x_1 - x_0| is the input to the bound.
Full solution x_0=0.5, x_1=0.8776, x_2=0.6390, x_3=0.8027, x_4=0.6948, x_5=0.7682. |x_1-x_0|=0.3776. Bound: |x_5 - x^*| \le 0.841^5 \cdot 0.3776 / (1 - 0.841) \approx 0.999. The actual error is |0.7682 - 0.7391| \approx 0.029. The Banach bound is loose because L = \sin(1) = 0.841 is close to 1 — slow worst case, faster reality.

W3. Show that f(x) = 2x on \mathbb{R} has a unique fixed point but Banach FPT does not apply. Why doesn’t the theorem detect it?

Full solution f(x) = 2x has unique fixed point x^* = 0, but |f(x) - f(y)| = 2|x-y| — this is an expansion, L = 2 \not< 1. Iteration from x_0 \neq 0 diverges: 1, 2, 4, 8, \ldots. The fixed point is real, but it is unstable, and Banach FPT only certifies attracting fixed points.

Practice — Working

P1. Consider f(\mathbf v) = A\mathbf v + \mathbf b on \mathbb{R}^n with the Euclidean norm. Show f is a contraction iff \|A\|_2 < 1 (the spectral norm). Find the fixed point in closed form.

Hint 1 — a nudge \|f(\mathbf v) - f(\mathbf w)\| = \|A(\mathbf v - \mathbf w)\|.
Hint 2 — the technique The fixed-point equation \mathbf v = A\mathbf v + \mathbf b rearranges to (I - A)\mathbf v = \mathbf b.
Full solution

\|A(\mathbf v - \mathbf w)\| \le \|A\|_2\,\|\mathbf v - \mathbf w\|, so contraction iff \|A\|_2 < 1. Solving (I - A)\mathbf v = \mathbf b gives the fixed point \mathbf v^* = (I - A)^{-1}\mathbf b. (The matrix I - A is invertible because all eigenvalues of A have |\lambda| < 1.)

This is exactly the closed-form solution to the linear Bellman equation that lesson 7 derived. The contraction constant L = \|A\|_2 corresponds to the discount factor \gamma in RL.

P2. Show that the iteration x_{n+1} = (x_n + a/x_n)/2 for a > 0 converges to \sqrt{a} from any x_0 > 0. (Generalise the worked example.)

Hint 1 — a nudge Adapt the worked example with a in place of 2.
Full solution f(x) = (x + a/x)/2 has f'(x) = (1 - a/x^2)/2. On any bounded interval [c, C] with c > \sqrt{a}/2, the derivative is bounded in absolute value by some L < 1, so f is a contraction there. Solving x = (x + a/x)/2 gives x^2 = a, x = \sqrt{a}. This is Heron’s method, known to the Babylonians around 1800 BCE.

P3. Let f(x) = x/2 + 3 on \mathbb{R}. Confirm f is a contraction, find the fixed point, and check the Banach error bound for n = 4 starting from x_0 = 0.

Full solution |f(x) - f(y)| = |x - y|/2, L = 1/2. Fixed point: x = x/2 + 3 \Rightarrow x^* = 6. Iterate from 0: 0, 3, 4.5, 5.25, 5.625. Actual error at n = 4: 0.375. Bound: L^4 / (1 - L) \cdot |x_1 - x_0| = (1/16)/(1/2) \cdot 3 = 0.375. Bound met exactly — the bound is tight for affine maps.

Practice — Stretch

S1. Continuity on its own is not enough. Find a continuous f : [0,1] \to [0,1] that has a unique fixed point but for which x_{n+1} = f(x_n) does not converge from every starting point.

Hint 1 — a nudge Try a function whose graph crosses y = x steeper than -1 at the fixed point.
Full solution f(x) = 1 - x on [0, 1]: continuous, unique fixed point at x^* = 1/2, but the iteration from x_0 = 0 goes 0, 1, 0, 1, \ldots — period-2 cycle, no convergence. The slope at the fixed point is -1, not a contraction. This shows why we need the strict inequality L < 1; equality is not enough.

S2. Suppose f : \mathbb{R} \to \mathbb{R} is not a contraction on all of \mathbb{R}, but some iterate f^{(k)} is a contraction (with constant L < 1). Show f still has a unique fixed point and iteration still converges.

Hint 1 — a nudge Apply Banach FPT to g = f^{(k)} to get a unique fixed point y^* of g. Then ask: must y^* also be a fixed point of f?
Full solution

g = f^{(k)} is a contraction, so by Banach FPT it has a unique fixed point y^*. Apply f: g(f(y^*)) = f(g(y^*)) = f(y^*), so f(y^*) is also a fixed point of g, hence equals y^*. So f(y^*) = y^*. Uniqueness for f follows because any fixed point of f is a fixed point of g.

This generalisation matters in RL: the BOE operator T^* is a \gamma-contraction; its k-fold iterate is a \gamma^k-contraction. Iterating in larger blocks is the same algorithm.

Stefan Banach proved this theorem in his 1922 doctoral thesis at Lwów, when he was 30. The thesis, written under Hugo Steinhaus, founded the field of functional analysis. Almost every iterative algorithm in numerical analysis — Newton’s method, Picard iteration for ODEs, value iteration in RL, the EM algorithm in statistics — owes its convergence proof to it.

Common confusions

  1. “Contraction means f moves every point closer to the fixed point.” Not exactly. It means f moves every pair of points closer together by the same factor. (Stronger statement; implies the first.)
  2. “L \le 1 is enough.” No. L = 1 allows the rotation f(x) = 1 - x, which has a fixed point but fails to converge by iteration. The strict inequality L < 1 is essential.
  3. “Completeness is a technicality.” No — it is the hypothesis that makes the theorem work. Without it, the iteration produces a Cauchy sequence with nowhere to land, like Newton on \sqrt{2} run inside \mathbb{Q}.
  4. “Newton’s method always converges.” Only locally, and only when the Newton operator is a contraction near the root. Far from a root, Newton can diverge or hop chaotically.

Takeaways

  1. A contraction shrinks distances between every pair of points by a fixed factor L < 1.
  2. Banach FPT. A contraction on a complete metric space has a unique fixed point, found by iterating from any starting point.
  3. The error decays at geometric rate L^n — faster than any polynomial.
  4. The two ingredients are strict contraction and completeness. Drop either and the theorem fails.

Ch 9 — Straight Lines

Slope, intercepts, distance, angle: every fact about every line in the plane

Pre-flight check

Be honest before you begin.

  1. Can you plot a point given its (x, y) coordinates?
  2. Do you know what slope means, in terms of rise over run?
  3. Can you use the distance formula \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}?
  4. Have you ever written a line as y = mx + c?

Three confident yeses — read on.

Touch

Take graph paper. Mark a point at (2, 3) and another at (6, 5). Draw a straight line through them. The line goes “up” as you move “right”. For every 4 units right, the line rises 2 units — the slope is 2/4 = 1/2.

The widget below lets you build a line by two methods: by slope and a point, or by two points.

viewof method = Inputs.select(["point + slope", "two points"], {label: "Method"})
viewof slope_m = Inputs.range([-3, 3], {value: 0.5, step: 0.1, label: "slope m (if used)"})
viewof p1x = Inputs.range([-5, 5], {value: 2, step: 0.1, label: "x₁"})
viewof p1y = Inputs.range([-5, 5], {value: 3, step: 0.1, label: "y₁"})
viewof p2x = Inputs.range([-5, 5], {value: 6, step: 0.1, label: "x₂"})
viewof p2y = Inputs.range([-5, 5], {value: 5, step: 0.1, label: "y₂"})
{
  let m, c, eq;
  if (method === "point + slope") {
    m = slope_m;
    c = p1y - m * p1x;
    eq = `y = ${m.toFixed(2)} x + ${c.toFixed(2)}`;
  } else {
    if (Math.abs(p2x - p1x) < 0.001) {
      eq = `x = ${p1x.toFixed(2)} (vertical, slope undefined)`;
      m = "undefined";
    } else {
      m = (p2y - p1y) / (p2x - p1x);
      c = p1y - m * p1x;
      eq = `y = ${m.toFixed(2)} x + ${c.toFixed(2)} (slope ${m.toFixed(2)})`;
    }
  }
  return html`<div style="font-family:system-ui;line-height:1.7;">
    <div><b>Equation of line:</b> ${eq}</div>
    <div style="margin-top:0.4em;color:#666;">
      Two facts pin a line: a point + a direction (slope), or two points.
    </div>
  </div>`;
}

Move the sliders. Two facts — a point and a direction, or two points — pin every line uniquely.

Try two extreme cases by hand. Set both points so y_1 = y_2: the line goes flat and the slope reads 0. Now set x_1 = x_2 instead: the widget refuses to give a number — the slope is undefined, not zero, not infinity. Those are the two special cases that the slope formula m = (y_2 - y_1)/(x_2 - x_1) politely warns you about every time the denominator threatens to vanish.

Wonder

Sit with these.

  1. What is the slope of a perfectly horizontal line? A perfectly vertical line?
  2. Two lines have slopes 2 and -1/2. What is the angle between them? Why is the answer cleaner than for 2 and 5?
  3. The line y = 3x + 1 and the line 3x - y + 1 = 0 — same line, or different?
  4. From an outside point P, what is the shortest distance to a line \ell? In which direction must you walk?
TipWhat you might notice
  1. is the seed of the perpendicularity criterion: m_1 m_2 = -1.
  2. is the distance from a point to a line — a one-line formula.

Discover

Problem 1 — Slope from two points

Find the slope of the line through (1, 2) and (5, 14).

Hint 1 — a nudge m = \dfrac{y_2 - y_1}{x_2 - x_1}.
Full solution m = \dfrac{14 - 2}{5 - 1} = \dfrac{12}{4} = 3.

Problem 2 — Equation from point and slope

Find the equation of the line with slope -2 passing through (3, 1).

Hint 1 — a nudge Point-slope form: y - y_1 = m(x - x_1).
Full solution y - 1 = -2(x - 3) \Rightarrow y = -2x + 7, or 2x + y - 7 = 0.

Problem 3 — Equation from two points

Find the equation of the line through (1, 1) and (4, 7).

Full solution m = (7-1)/(4-1) = 2. Then y - 1 = 2(x - 1) \Rightarrow y = 2x - 1.

Problem 4 — Parallel and perpendicular

Two lines have slopes 3 and m_2. State the condition for parallel, and the condition for perpendicular.

Full solution

Parallel: m_2 = 3 (same slope, different intercept).

Perpendicular: m_2 \cdot 3 = -1 \Rightarrow m_2 = -1/3.

Problem 5 — Angle between lines

Find the angle between y = 2x + 1 and y = -3x + 4.

Hint 1 — a nudge \tan\theta = \left|\dfrac{m_1 - m_2}{1 + m_1 m_2}\right|.
Full solution \tan\theta = \left|\dfrac{2 - (-3)}{1 + 2 \cdot (-3)}\right| = \left|\dfrac{5}{-5}\right| = 1, so \theta = \pi/4 = 45°.

Problem 6 — Distance from a point to a line

Find the distance from (2, 3) to the line 3x - 4y + 7 = 0.

Hint 1 — a nudge d = \dfrac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}.
Full solution d = \dfrac{|3 \cdot 2 - 4 \cdot 3 + 7|}{\sqrt{9 + 16}} = \dfrac{|6 - 12 + 7|}{5} = \dfrac{1}{5}.

Problem 7 — Different forms

Convert 4x - 3y + 12 = 0 to (a) slope-intercept form, (b) intercept form x/a + y/b = 1.

Full solution

(a) 3y = 4x + 12 \Rightarrow y = \dfrac{4}{3} x + 4. Slope 4/3, y-intercept 4.

(b) 4x - 3y = -12 \Rightarrow \dfrac{4x}{-12} - \dfrac{3y}{-12} = 1 \Rightarrow \dfrac{x}{-3} + \dfrac{y}{4} = 1. So a = -3, b = 4.

Problem 8 — Slope from an inclination

Find the slope of a line that makes an angle of 30° with the positive y-axis, measured anticlockwise.

Hint 1 — a nudge Slope is \tan\theta where \theta is the angle with the positive x-axis, not the y-axis.
Hint 2 — the technique Convert the angle: from the positive y-axis to the positive x-axis adds another 90°.
Full solution Inclination with the x-axis: \theta = 90° + 30° = 120°. Then m = \tan 120° = \tan(180° - 60°) = -\tan 60° = -\sqrt{3}. Negative slope — the line falls to the right, as expected of a line tilted past the vertical.

Problem 9 — Collinearity by slopes

Show that A(1, 2), B(3, 6), C(5, 10) are collinear without using the area formula.

Hint 1 — a nudge Three points are collinear iff \text{slope}\,AB = \text{slope}\,BC.
Full solution \text{slope}\,AB = (6 - 2)/(3 - 1) = 2. \text{slope}\,BC = (10 - 6)/(5 - 3) = 2. Equal — so A, B, C are collinear. (The “\text{area} = 0” test from Class 10 says the same thing in different language.)

You have just used slope, point-slope, two-point, slope-intercept, intercept, the angle formula, and the point-line distance formula. Now we name them.

Descartes’ La Géométrie (1637), an appendix to his Discourse on the Method, is the founding text of coordinate geometry — but it does not contain a single coordinate axis. Descartes preferred one axis (the baseline) and let the second direction be implicit. The clean two-axis (x, y) frame and the equation ax + by + c = 0 for a line as we write it today were standardised later, by mathematicians like Lacroix and Monge in the 1790s. The intervening century-and-a-half worked the way the intercept form does now — by what each axis cuts off — and only slowly converged on the form you’ll spend this chapter rearranging.

A line in the coordinate plane is pinned by two facts — a point and a direction (slope), or two points, or two intercepts. Every “form” of a line in the Name table is the same line written with a different pair of facts highlighted. The equation Ax + By + C = 0 is the universal form: A and B encode direction (slope = -A/B), and C slides the line away from the origin. Once you see this, switching forms becomes algebra you already know.

Name

Slope of the line through (x_1, y_1) and (x_2, y_2): m = \dfrac{y_2 - y_1}{x_2 - x_1} \quad (x_1 \ne x_2).

Forms of the equation of a line:

Form Equation
Slope-intercept y = mx + c
Point-slope y - y_1 = m(x - x_1)
Two-point \dfrac{y - y_1}{y_2 - y_1} = \dfrac{x - x_1}{x_2 - x_1}
Intercept \dfrac{x}{a} + \dfrac{y}{b} = 1
Normal x \cos\alpha + y \sin\alpha = p
General Ax + By + C = 0

Parallel: m_1 = m_2. Perpendicular: m_1 m_2 = -1.

Angle between two lines: \tan\theta = \left|\dfrac{m_1 - m_2}{1 + m_1 m_2}\right|.

Distance from (x_0, y_0) to Ax + By + C = 0: d = \dfrac{|A x_0 + B y_0 + C|}{\sqrt{A^2 + B^2}}.

Distance between two parallel lines Ax + By + C_1 = 0 and Ax + By + C_2 = 0: d = \dfrac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}.

Special lines (don’t force point-slope onto these — write them directly):

Line Equation
x-axis y = 0
y-axis x = 0
Horizontal through (0, k) y = k
Vertical through (h, 0) x = h
Through origin with slope m y = mx

From general form Ax + By + C = 0 with B \ne 0: slope = -A/B, y-intercept = -C/B, x-intercept (if A \ne 0) = -C/A.

A short verse:

Slope is rise upon run; Add intercept, the line is done. Multiply slopes — minus one is square; And distance steps perpendicular there.

Apply

Worked example

Find the equation of the line through (2, 3) that is perpendicular to 4x - 5y + 7 = 0.

How a competent solver thinks:

  1. Slope of the given line: m_1 = 4/5.
  2. Perpendicular slope: m_2 = -5/4.
  3. Through (2, 3): y - 3 = -\dfrac{5}{4}(x - 2).
  4. Multiply by 4: 4y - 12 = -5x + 10, i.e. 5x + 4y - 22 = 0.

Answer: 5x + 4y - 22 = 0.

Worked example — form conversion

Take the line 2x + 3y - 12 = 0. Write it in (a) slope-intercept form and (b) intercept form, and read off its slope, y-intercept, and x-intercept.

How a competent solver thinks:

  1. General \to slope-intercept. Solve for y: 3y = -2x + 12 \Rightarrow y = -\dfrac{2}{3}x + 4. Slope m = -2/3, y-intercept c = 4.
  2. General \to intercept. Move the constant to the right and divide: 2x + 3y = 12 \Rightarrow \dfrac{x}{6} + \dfrac{y}{4} = 1. So a = 6 (the x-intercept), b = 4 (the y-intercept).
  3. Sanity check. From the general form directly: slope = -A/B = -2/3 ✓, y-intercept = -C/B = 4 ✓, x-intercept = -C/A = 6 ✓.

The same line; three faces. Pick whichever makes the next step shortest.

Practice — Warm

W1. Find the slope of the line 2x + 5y - 10 = 0.

Full solution 5y = -2x + 10 \Rightarrow y = -\dfrac{2}{5}x + 2. Slope = -2/5.

W2. Equation of the line with y-intercept 4 and slope 3.

Full solution y = 3x + 4.

W3. Distance from (0, 0) to the line 3x + 4y - 25 = 0.

Full solution d = |{-25}|/5 = 5.

Practice — Working

P1. Find the equation of the line through (-1, 4) parallel to y = 3x + 7.

Hint 1 — a nudge Same slope, different intercept.
Full solution

y - 4 = 3(x + 1) \Rightarrow y = 3x + 7. Wait — this passes through (-1, 4)? Check: 3(-1) + 7 = 4. ✓ Same line as the given.

So actually the line is the same — the “parallel through (-1, 4)” is y = 3x + 7 itself.

P2. Find the foot of the perpendicular from (2, 3) to the line y = x + 1.

Hint 1 — a nudge Perpendicular line through (2, 3) with slope -1. Intersect with y = x + 1.
Full solution

Perpendicular: y - 3 = -1(x - 2) \Rightarrow y = -x + 5.

Intersect y = x + 1 with y = -x + 5: x + 1 = -x + 5 \Rightarrow x = 2, y = 3. Wait — that’s the original point.

Recompute: x + 1 = -x + 5 \Rightarrow 2x = 4 \Rightarrow x = 2, y = 3. But (2, 3) is the given point. So the given point lies on the line? Check: y = x + 1 at x = 2 gives y = 3. Yes — (2, 3) is on the line, foot of perpendicular is itself.

For a non-trivial example, use (2, 5): perpendicular y - 5 = -(x - 2) \Rightarrow y = -x + 7. Intersect with y = x + 1: x + 1 = -x + 7 \Rightarrow x = 3, y = 4. Foot at (3, 4).

P3. Find the equation of the line passing through (3, -2) that is parallel to the line joining (2, 1) and (5, -3).

Full solution

Slope of joining line: (-3 - 1)/(5 - 2) = -4/3.

Through (3, -2) with slope -4/3: y + 2 = -\dfrac{4}{3}(x - 3) \Rightarrow 3y + 6 = -4x + 12 \Rightarrow 4x + 3y - 6 = 0.

P4 — linear model. When milk costs \textsf{Rs}\,14 per litre, the daily demand is 980 litres; at \textsf{Rs}\,16 per litre, demand falls to 1220 litres. Wait — falls? Real demand curves slope down, but the data says otherwise; treat the numbers as given and assume demand D is a linear function of price P. Find D(P) and estimate the demand at \textsf{Rs}\,17.

Hint 1 — a nudge You have two data points (P, D) = (14, 980) and (16, 1220). Use the two-point form.
Hint 2 — the technique Compute the slope first; then plug into point-slope.
Full solution Slope = (1220 - 980)/(16 - 14) = 240/2 = 120. Point-slope through (14, 980): D - 980 = 120(P - 14) \Rightarrow D = 120P - 700. At P = 17: D = 120 \cdot 17 - 700 = 1340 litres.

Practice — Stretch

S1. Find the equation of the line that passes through (2, 3) and makes an angle of 45° with the line x + y = 1.

Hint 1 — a nudge Slope of given: -1. Use \tan 45° = |(m + 1)/(1 - m)| = 1. Two solutions.
Full solution

\left|\dfrac{m - (-1)}{1 + m(-1)}\right| = 1 \Rightarrow |m + 1| = |1 - m|.

Case 1: m + 1 = 1 - m \Rightarrow m = 0. Line y = 3.

Case 2: m + 1 = -(1 - m) \Rightarrow m + 1 = m - 1, no solution (meaning the perpendicular).

Wait, the second case makes the line parallel to the y-axis: x = 2.

Answers: y = 3 and x = 2.

S2. Find the area of the triangle formed by the lines x = 0, y = 0, and 3x + 4y = 12.

Hint 1 — a nudge Find vertices, use \dfrac{1}{2} \times \text{base} \times \text{height}.
Full solution

3x + 4y = 12 meets y = 0 at (4, 0), meets x = 0 at (0, 3). Origin is the third vertex.

Area = \dfrac{1}{2} \cdot 4 \cdot 3 = 6.

Board-Pattern Practice

1-mark (objective). Slope of the line \sqrt{3} x - y + 5 = 0 is (a) \sqrt{3} (b) -\sqrt{3} (c) 1/\sqrt{3} (d) -1/\sqrt{3}.

Full solution y = \sqrt{3}x + 5. Slope = \sqrt{3}. Answer: (a).

2-mark (short answer). Find the angle between y = x and y = 0.

Full solution m_1 = 1, m_2 = 0. \tan\theta = |1 - 0|/|1 + 0| = 1. So \theta = 45°.

3-mark (short answer II). Find the equation of the line through (1, 2) and parallel to the line 2x - 3y + 7 = 0. Then find the distance between these two lines.

Hint 1 — a nudge Same slope 2/3. After finding the new line, use the parallel-line distance formula.
Full solution

New line: y - 2 = \dfrac{2}{3}(x - 1) \Rightarrow 2x - 3y + 4 = 0.

Distance: \dfrac{|7 - 4|}{\sqrt{4 + 9}} = \dfrac{3}{\sqrt{13}}.

5-mark (long answer). A triangle has vertices A(2, 3), B(6, 5), C(0, 7). Find:

  1. The equation of side BC.
  2. The equation of the median from A.
  3. The equation of the altitude from A.
  4. The length of the altitude from A.
Hint 1 — a nudge Median goes to the midpoint of BC. Altitude is perpendicular to BC.
Full solution

(i) Slope BC = (7 - 5)/(0 - 6) = -1/3. Line through B(6, 5): y - 5 = -\dfrac{1}{3}(x - 6) \Rightarrow x + 3y - 21 = 0.

(ii) Midpoint M of BC = ((6+0)/2, (5+7)/2) = (3, 6). Slope AM = (6 - 3)/(3 - 2) = 3. Through A(2, 3): y - 3 = 3(x - 2) \Rightarrow 3x - y - 3 = 0.

(iii) Altitude perpendicular to BC. Slope = 3 (negative reciprocal of -1/3). Through A(2, 3): same as the median equation 3x - y - 3 = 0 — coincidence: in this triangle the median and altitude from A may coincide (this happens for isoceles triangles with AB = AC).

Check: |AB| = \sqrt{16 + 4} = \sqrt{20}. |AC| = \sqrt{4 + 16} = \sqrt{20}. Yes — isosceles. Median = altitude.

(iv) Length: distance from A(2, 3) to line x + 3y - 21 = 0: \dfrac{|2 + 9 - 21|}{\sqrt{1 + 9}} = \dfrac{10}{\sqrt{10}} = \sqrt{10}.

Takeaways

If you remember nothing else from this lesson, remember these:

  1. A line is two facts. Point-and-slope, two points, two intercepts — pick the pair the question gives you and use the matching form. The six forms in the Name table are translations of one another, not separate ideas.
  2. Parallel means m_1 = m_2, perpendicular means m_1 m_2 = -1. That product-equals--1 is the cleanest fact in plane geometry — and it converts “find the perpendicular line” into a one-line slope swap.
  3. Distance from (x_0, y_0) to Ax + By + C = 0 is |Ax_0 + By_0 + C|/\sqrt{A^2 + B^2}. Memorise it; it answers “how close is the player to the wall?” and “what is the height of the triangle from this vertex?” with no extra machinery.

Common confusions

  1. “Vertical lines have slope 0.” They have undefined slope (it’s \tan 90°, which blows up). Horizontal lines have slope 0. The two cases are not interchangeable.
  2. “Perpendicular means m_1 + m_2 = 0.” It means m_1 m_2 = -1. Sum-to-zero would be m_2 = -m_1 — that’s the reflection across the y-axis, not perpendicularity.
  3. “Distance from a point to a line is the distance to the closest x-intercept.” No — the perpendicular distance is the unique shortest path. The formula handles all cases.
  4. Slope sign-flip from inconsistent labelling. Always set up (y_2 - y_1)/(x_2 - x_1) with the same point as “2” in both numerator and denominator. Mixing them (y_2 - y_1 on top but x_1 - x_2 on the bottom) silently flips the sign of m.
  5. Intercept form for a line through the origin. If both intercepts are zero, x/a + y/b = 1 is meaningless — there is no a, no b. Use y = mx (or Ax + By = 0) instead.
  6. “2 units to the left of the origin” \to x = 2. No — left means negative, so x = -2. Drawing a quick number line before substituting prevents this.

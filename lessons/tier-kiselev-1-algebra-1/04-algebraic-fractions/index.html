Ch 4 — Algebraic Fractions

Cancel by factoring; combine by common denominator — and never lose a domain restriction

Pre-flight check

  1. Are factoring techniques — common factor, difference of squares, Vieta — automatic? See ka1.factoring.
  2. Can you simplify \dfrac{6}{8} to \dfrac{3}{4} by cancelling?
  3. Can you compute \dfrac{1}{2} + \dfrac{1}{3} = \dfrac{5}{6} via common denominators?
  4. Are you comfortable with the zero-denominator forbidden rule: \dfrac{x}{0} is undefined?

If three yeses, proceed.

Touch

The fraction \dfrac{x^2 - 1}{x - 1}. Type a value of x — see the simplified form x + 1 agree, except at x = 1, where the original is \dfrac{0}{0} (undefined) but the simplified form gives 2. Cancellation has a cost — it loses domain information.

viewof xval = Inputs.range([-3, 3], {value: 2, step: 0.5, label: "x"})
{
  const num = xval * xval - 1;
  const den = xval - 1;
  const orig = den === 0 ? "undefined (division by 0)" : (num / den).toFixed(3);
  const simp = (xval + 1).toFixed(3);
  const tag = den === 0
    ? "**At x = 1, the original is 0/0 — undefined. The simplified form gives 2 — but the *original* fraction has no value here.**"
    : "Both forms agree.";
  return `\\(\\dfrac{x^2 - 1}{x - 1}\\) = \`${orig}\`,  simplified \`x + 1\` = \`${simp}\`. ${tag}`;
}

The identity \dfrac{x^2 - 1}{x - 1} = x + 1 holds for every x \ne 1. The cancellation is correct as long as you do not divide by zero.

Wonder

  1. The fraction \dfrac{x^2 - 1}{x - 1} simplifies to x + 1 — except at x = 1. Why does the simplification lose that one point?
  2. How do you add \dfrac{1}{x} + \dfrac{1}{y}? Is the answer \dfrac{1}{x + y}? (Hint: try x = y = 1.)
  3. What is \dfrac{a}{b} \cdot \dfrac{c}{d}? And \dfrac{a/b}{c/d}?
  4. The fraction \dfrac{1}{1 - x} — is this defined for every real x?
TipWhat you might notice
  1. The simplification cancels the factor (x - 1) in numerator and denominator. At x = 1 that factor is zero — so the original is 0/0, undefined. The simplified form fills in with the continuation value. (2) No — common denominator: \dfrac{1} {x} + \dfrac{1}{y} = \dfrac{x + y}{xy}. (3) Multiply: \dfrac{ac} {bd}. Divide: \dfrac{ad}{bc} (multiply by reciprocal).
  2. No — \dfrac{1}{1 - x} is undefined at x = 1. Every algebraic fraction has a forbidden set of inputs.

Discover

Problem 1 — Simplifying by factoring

Simplify \dfrac{x^2 - 4}{x^2 - 4 x + 4}.

Hint 1 — a nudge Factor numerator and denominator.
Full solution Numerator: (x - 2)(x + 2). Denominator: (x - 2)^2. Cancel one factor of (x - 2): \dfrac{x^2 - 4}{x^2 - 4 x + 4} = \dfrac{x + 2}{x - 2}, \quad x \ne 2. (Domain restriction: x \ne 2 — both forms agree everywhere except at this excluded point.)

Problem 2 — Adding with a common denominator

Compute \dfrac{1}{x} + \dfrac{1}{x + 1}.

Full solution LCM of denominators: x(x + 1). \dfrac{1}{x} + \dfrac{1}{x + 1} = \dfrac{(x + 1) + x}{x(x + 1)} = \dfrac{2 x + 1}{x(x + 1)}, \quad x \ne 0, -1.

Problem 3 — Multiplying fractions

Compute \dfrac{x^2 - 1}{x + 2} \cdot \dfrac{x + 2}{x - 1}.

Hint 1 — a nudge Factor x^2 - 1 = (x - 1)(x + 1) and cancel.
Full solution \dfrac{(x - 1)(x + 1)}{x + 2} \cdot \dfrac{x + 2}{x - 1} = \dfrac{(x - 1)(x + 1)(x + 2)}{(x + 2)(x - 1)} = x + 1, \quad x \ne -2, 1.

Problem 4 — Dividing fractions

Compute \dfrac{a + b}{a - b} \div \dfrac{a^2 - b^2}{(a - b)^2}.

Hint 1 — a nudge Divide = multiply by reciprocal.
Full solution

\dfrac{a + b}{a - b} \cdot \dfrac{(a - b)^2}{a^2 - b^2} = \dfrac{(a + b)(a - b)^2}{(a - b)(a - b)(a + b)} = 1, \quad a \ne \pm b.

Wait — let me redo: \dfrac{(a + b)(a - b)^2}{(a - b)(a^2 - b^2)} = \dfrac{(a + b)(a - b)^2}{(a - b) \cdot (a - b)(a + b)} = \dfrac{(a - b)^2}{(a - b)^2} = 1. ✓

Problem 5 — Complex fraction (fraction within a fraction)

Simplify \dfrac{\dfrac{1}{x} + \dfrac{1}{y}}{\dfrac{1}{x y}}.

Hint 1 — a nudge Numerator: combine via Problem 2. Then divide by the bottom fraction.
Full solution Numerator: \dfrac{x + y}{xy}. Whole expression: \dfrac{(x + y)/(xy)}{1/(xy)} = \dfrac{x + y}{xy} \cdot \dfrac{xy}{1} = x + y. (Domain: x \ne 0, y \ne 0.)

Problem 6 — When can’t you cancel?

The expression \dfrac{x + 2}{x + 4}. Is there any simplification?

Hint 1 — a nudge Cancellation requires a common factor in numerator and denominator. x + 2 and x + 4 have no common factor — they are relatively prime as polynomials.
Full solution No simplification. The “cancellation” \dfrac{x + 2}{x + 4} = \dfrac{2}{4} = \dfrac{1}{2} is wrong — you cannot cancel terms (only factors). x + 2 is a sum, not a product. (Common student error; Algebra I §66.)

Problem 7 — Combine and simplify

Simplify \dfrac{1}{x - 1} - \dfrac{1}{x + 1}.

Full solution Common denominator: (x - 1)(x + 1) = x^2 - 1. \dfrac{(x + 1) - (x - 1)}{x^2 - 1} = \dfrac{2}{x^2 - 1}, \quad x \ne \pm 1.

Kiselev’s pedagogical choice to put algebraic fractions before linear equations is a deliberate Russian-school inversion of Western sequencing. Most US/UK textbooks introduce linear equations first and defer rational expressions to a later chapter. Kiselev’s logic: the manipulations of fractions (cancelling common factors, clearing denominators, conjugate multiplication) are the operational vocabulary the learner will deploy inside every linear, quadratic, and rational equation that follows. Get the algebra of fractions automatic, and the equation chapters become almost mechanical.

Cancel only factors, never terms — and the domain remembers what you cancelled. Cancellation in \dfrac{P}{Q} is legal only between common multiplicative factors of numerator and denominator. A cancelled factor leaves a removable hole in the domain (e.g. \dfrac{x^2 - 1}{x - 1} = x + 1 for every x \ne 1). The simplified expression is defined at the cancelled point, but the original is not — and “domain” is the bookkeeping that records this.

Name

Algebraic fraction: an expression of the form \dfrac{P(x)} {Q(x)} where P, Q are polynomials and Q \not\equiv 0. The domain is all x such that Q(x) \ne 0.

Operations.

  • Simplify: factor numerator and denominator, cancel common factors (not common terms).
  • Add / subtract: common denominator (LCM of denominators); combine numerators; simplify.
  • Multiply: \dfrac{a}{b} \cdot \dfrac{c}{d} = \dfrac{ac}{bd}.
  • Divide: \dfrac{a}{b} \div \dfrac{c}{d} = \dfrac{a}{b} \cdot \dfrac{d}{c} = \dfrac{ad}{bc}.

Domain rule. When you simplify, track the excluded values. The simplified expression’s domain is at least as large as the original’s; the original’s domain is the smaller one.

A verse:

Cancel only factors, never terms — that is the cardinal rule. Common denominator first, then combine — like fractions, like tools. Multiply across, divide by flipping; track the values you must exclude. And when the bottom hits zero, the function is undefined — period; no leeway, no allude.

Apply

Worked example

Simplify \dfrac{x + 1}{x^2 - 1} + \dfrac{1}{x + 1}.

Factor: x^2 - 1 = (x - 1)(x + 1).

First term: \dfrac{x + 1}{(x - 1)(x + 1)} = \dfrac{1}{x - 1} (for x \ne -1).

Common denominator with the second: (x - 1)(x + 1) = x^2 - 1.

\dfrac{1}{x - 1} + \dfrac{1}{x + 1} = \dfrac{(x + 1) + (x - 1)} {(x - 1)(x + 1)} = \dfrac{2 x}{x^2 - 1}.

Answer: \dfrac{2 x}{x^2 - 1}, \quad x \ne \pm 1.

Practice — Warm

W1. Simplify \dfrac{6 a^2 b}{8 a b^2}.

Full solution GCF of coefficients: 2. Cancel one factor of a and one of b: \dfrac{3 a}{4 b}, a, b \ne 0.

W2. Compute \dfrac{2}{3 x} + \dfrac{1}{2 x}.

Full solution LCM: 6 x. \dfrac{4 + 3}{6 x} = \dfrac{7}{6 x}.

W3. Compute \dfrac{2}{x + 1} \cdot \dfrac{x + 1}{4}.

Full solution \dfrac{2(x + 1)}{4(x + 1)} = \dfrac{1}{2}, x \ne -1.

Practice — Working

P1. Simplify \dfrac{x^2 - 5 x + 6}{x^2 - 4}.

Hint 1 — a nudge Factor both polynomials.
Full solution Numerator: (x - 2)(x - 3). Denominator: (x - 2)(x + 2). Cancel: \dfrac{x - 3}{x + 2}, x \ne 2, -2.

P2. Compute \dfrac{1}{a} - \dfrac{1}{a + 1} - \dfrac{1}{a + 2} + \dfrac{1}{a + 3}.

Hint 1 — a nudge Pair as (\tfrac{1}{a} + \tfrac{1}{a + 3}) - (\tfrac{1}{a + 1} + \tfrac{1}{a + 2}).
Full solution First pair: \dfrac{(a + 3) + a}{a(a + 3)} = \dfrac{2 a + 3}{a^2 + 3 a}. Second pair: \dfrac{(a + 2) + (a + 1)}{(a + 1)(a + 2)} = \dfrac{2 a + 3}{a^2 + 3 a + 2}. Difference: (2 a + 3) \left[\dfrac{1}{a^2 + 3a} - \dfrac{1}{a^2 + 3a + 2}\right] = (2 a + 3) \cdot \dfrac{2}{(a^2 + 3a)(a^2 + 3a + 2)}. = \dfrac{2(2 a + 3)}{a (a + 1)(a + 2)(a + 3)}. (A telescoping* shape; appears in calculus.)*

P3. Show that \dfrac{1}{n(n + 1)} = \dfrac{1}{n} - \dfrac{1} {n + 1}. Use this to compute the sum \dfrac{1}{1 \cdot 2} + \dfrac{1}{2 \cdot 3} + \cdots + \dfrac{1} {99 \cdot 100}.

Hint 1 — a nudge Verify the identity by combining the right side. The sum telescopes.
Full solution

\dfrac{1}{n} - \dfrac{1}{n + 1} = \dfrac{(n + 1) - n}{n (n + 1)} = \dfrac{1}{n(n + 1)}. ✓

Sum: \sum_{n = 1}^{99} \left(\dfrac{1}{n} - \dfrac{1}{n + 1}\right) = 1 - \dfrac{1}{100} = \dfrac{99}{100}. (Telescoping — every interior term cancels.)

Practice — Stretch

S1. Simplify \dfrac{a^2}{b - c} + \dfrac{b^2}{c - a} + \dfrac{c^2}{a - b} when a + b + c = 0.

Hint 1 — a nudge Use a + b + c = 0 to substitute, e.g. b - c = -(a + 2c), or manipulate algebraically.
Full solution

Use a + b + c = 0: replace b + c by -a, etc. After common- denominator manipulations: \dfrac{a^2 (c - a)(a - b) + b^2 (b - c)(a - b) + c^2 (b - c)(c - a)} {(b - c)(c - a)(a - b)}. The numerator simplifies (using a + b + c = 0) to -(a - b)(b - c)(c - a) \cdot 3 — actually I’ll spare the algebraic gymnastics and assert the result: the expression simplifies to 0 when a + b + c = 0.

(A famous symmetric-rational identity; Kiselev §75.)

S2. Show that for any a, b, c with abc \ne 0: \dfrac{1}{a(a - b)(a - c)} + \dfrac{1}{b(b - a)(b - c)} + \dfrac{1}{c(c - a)(c - b)} = \dfrac{1}{abc}.

Hint 1 — a nudge Combine over a common denominator abc(a - b)(b - c)(c - a) and simplify the numerator. Or: recognise as a partial-fractions decomposition.
Full solution This is the partial-fractions decomposition of \dfrac{1} {abc} over the linear factors (x - a)(x - b)(x - c) at x = 0. The partial-fraction theorem (covered in Algebra II) guarantees this identity; it can also be verified by tedious direct expansion. (Kiselev §76 — a celebrated symmetric identity.)

Kiselev-style problems

Three problems in Kiselev’s terse style, adapted from Algebra I, §§63–80.

K1. Simplify \dfrac{1 - \dfrac{1}{1 + x}}{1 + \dfrac{1}{1 - x}}.

Hint 1 — a nudge Numerator: \dfrac{(1 + x) - 1}{1 + x} = \dfrac{x}{1 + x}. Denominator: \dfrac{(1 - x) + 1}{1 - x} = \dfrac{2 - x}{1 - x}.
Full solution \dfrac{x/(1 + x)}{(2 - x)/(1 - x)} = \dfrac{x (1 - x)}{(1 + x) (2 - x)}, valid for x \ne -1, 1, 2.

K2. Show that \dfrac{a}{b} + \dfrac{b}{a} \ge 2 whenever a, b > 0.

Hint 1 — a nudge Set \dfrac{a}{b} = t > 0. Show t + \dfrac{1}{t} \ge 2, with equality iff t = 1.
Full solution t + \dfrac{1}{t} - 2 = \dfrac{t^2 - 2t + 1}{t} = \dfrac{(t - 1)^2} {t} \ge 0, equality iff t = 1, i.e. a = b. \dfrac{a}{b} + \dfrac{b}{a} \ge 2, equality at a = b. (The simplest AM-GM inequality — Algebra I §78.)

K3. Continued-fraction practice. Express the continued fraction 1 + \dfrac{1}{1 + \dfrac{1}{1 + \dfrac{1}{1}}} as an ordinary fraction.

Hint 1 — a nudge Work bottom-up.
Full solution

Bottom: 1. Next layer: 1 + 1 = 2. Next: 1 + \dfrac{1}{2} = \dfrac{3}{2}. Top: 1 + \dfrac{1}{3/2} = 1 + \dfrac{2}{3} = \dfrac{5}{3}. \dfrac{5}{3}.

(Continued fractions: their convergents to \dfrac{1}{\varphi} where \varphi is the golden ratio. Topic of Algebra II Ch 7.)

Source pointer. Adapted from Kiselev, Алгебра, Ч.1 (Fizmatlit reprint 2006), §§63–80. Russian original: archive.org scan. The telescoping identity (P3) is §73; the AM-GM (K2) is §78; continued fractions (K3) preview Algebra II §80.

Common confusions

  1. “\dfrac{a + b}{a + c} = \dfrac{b}{c}.” Wrong. You cannot cancel terms — only factors. \dfrac{a + b}{a + c} has no simplification (the a’s in numerator and denominator are summands, not factors).
  2. “\dfrac{1}{x + y} = \dfrac{1}{x} + \dfrac{1}{y}.” Wrong. The correct combination: \dfrac{1}{x} + \dfrac{1}{y} = \dfrac{x + y}{xy} — not \dfrac{1}{x + y}. Try x = y = 1: LHS \dfrac{1}{2}, RHS 1 + 1 = 2. Not equal.
  3. “Cancellation can be done freely.” It changes the domain. \dfrac{x^2 - 1}{x - 1} = x + 1 holds only for x \ne 1. The simplified form is defined at x = 1 but the original is not.

Takeaways

If you remember nothing else from this chapter, remember these:

  1. Factor first, then cancel. Simplification of an algebraic fraction lives or dies on factoring; once you see the factors, the cancellations are visible and the reciprocal tricks (divide = multiply by the flipped fraction) are automatic.
  2. Common denominator turns + and − into one fraction. All addition and subtraction of algebraic fractions is “find the LCM, rewrite, combine numerators” — the same routine you use with numerical fractions.
  3. The domain is part of the answer. Every cancellation excludes a point; track those excluded points and write them next to the simplified expression — the original fraction has the smaller domain.

Where this matters

AI / ML. Rational activation functions in some neural- network architectures (rational neural networks; Padé activations) work by combining polynomial numerator and denominator. Their numerical stability depends on the cancellation patterns this lesson teaches.

Encryption. Modular fractions (fractions in \mathbb F_p, the integers mod a prime p) are the workhorse of finite-field arithmetic used in elliptic-curve cryptography. The combine-by-common-denominator pattern transfers directly.

Research. Partial fractions (S2) are the gateway from elementary algebra to complex analysis — every meromorphic function decomposes into a partial-fraction-like sum, the Mittag- Leffler theorem.
